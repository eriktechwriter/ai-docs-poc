# Agent Control Settings

### Procedure {#procedure}

1.  Select **Global Agent Settings**.

2.  Click the **Agent Control** tab.

3.  Configure settings as required.

    <table>
    <colgroup>
    <col style="width: 25%" />
    <col style="width: 75%" />
    </colgroup>
    <thead>
    <tr>
    <th><p>Section</p></th>
    <th><p>Settings</p></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><p><strong>General Settings</strong></p></td>
    <td><p><strong>Add Manual Scan to the Windows shortcut menu on endpoints</strong>: Displays the <strong>Scan</strong> menu item to the Trend Vision One Endpoint Security agents shortcut menu</p></td>
    </tr>
    <tr>
    <td><p><strong>Alert Settings</strong></p></td>
    <td><ul>
    <li><p><strong>Show the alert icon on the Windows taskbar if the virus pattern file is not updated after XX day(s)</strong>: Displays an alert icon on the Windows taskbar to remind users to update an outdated Virus Pattern after the specified number of days</p></li>
    <li><p><strong>Display a notification message if the endpoint needs to restart to load a kernel mode driver</strong>: Displays a notification on the endpoint indicating that a restart is required to finish installing a hotfix or upgrade package that contains a new kernel mode driver</p></li>
    </ul></td>
    </tr>
    <tr>
    <td><p><strong>Agent Language Configuration</strong></p></td>
    <td><p>You can configure all Trend Vision One Endpoint Security agents to display using the Trend Vision One server language settings or the locally logged on user language settings.</p>
    <ul>
    <li><p><strong>Local language settings on the endpoint</strong>: The Trend Vision One Endpoint Security agent displays using the language settings of the logged on user.</p>


    :::note

    <p>If the Trend Vision One Endpoint Security agent does not support the logged on user language settings, the agent applies the Trend Vision One server language. If the endpoint does not support the Trend Vision One server language, English displays.</p>


    :::

    </li>
    <li><p><strong>Apex One server language</strong>: The Trend Vision One Endpoint Security agent displays using the Trend Vision One server language.</p>


    :::note

    <p>If the endpoint does not support the Trend Vision One server language, English displays.</p>


    :::

    </li>
    </ul></td>
    </tr>
    </tbody>
    </table>

4.  Click **Save**.
:::
