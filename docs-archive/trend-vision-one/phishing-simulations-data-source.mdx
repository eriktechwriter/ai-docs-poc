# Add phishing simulations as a data source

July 12, 2024 – You can now add Trend Vision One Phishing Simulations as a data source in the Threat and Exposure Management, which allows access to breach events from phishing simulations. For more information, see [Configurating data sources](configuring-data-sources.mdx).

**Cyber Risk Exposure Management → Threat and Exposure Management**
