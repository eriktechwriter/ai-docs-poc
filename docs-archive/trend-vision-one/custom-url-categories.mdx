# Custom URL categories

:Add custom URL categories to subject URLs that are not part of predefined categories to secure access rules.

Adding and configuring custom URL categories allows you to apply groups of URLs you define to rules and settings throughout **Zero Trust Secure Access**.

Custom URL categories can be applied to the following rules and settings:

- Risk rules

- Internet Access rules

- HTTPS inspection rules

- Bandwidth control rules

- Outbound static IP settings

Internet Access uses Trend Micro Web Reputation Services to categorize URLs.
