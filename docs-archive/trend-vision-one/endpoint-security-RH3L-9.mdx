# Endpoint Security Supports Red Hat Enterprise Linux 9 (ARM64)

June 25, 2025—Trend Vision One Endpoint Security now supports Red Hat Enterprise Linux 9 (ARM64). Both Server & Workload Protection and Endpoint Sensor now support deploying to Red Hat Enterprise Linux 9 (ARM64) endpoints. When [deploying a new agent in Endpoint Inventory](deploy-agents.mdx), you can now select Red Hat Enterprise Linux 9 (ARM64). For details on supported Linux platforms, see [Endpoint Agent System Requirements](endpoint-agent-system-requirements.mdx).

**Endpoint Security → Endpoint Inventory**
