# Virtual Network Sensor

:::warning[Important]

The Virtual Network Sensor only supports updates for versions released within the past 6 months. If your Virtual Network Sensor version is older than six months, you must redeploy to get the latest version.
:::

- July 2025

  - [Virtual Network Sensor version 1.0.1524](rn-vns-101524.mdx)

    Includes updates and enhances the capabilities of Virtual Network Sensor to categorize traffic that was previously unidentifiable.

- June 2025

  - [Virtual Network Sensor version 1.0.1504](rn-vns-101504.mdx)

    Includes updates and adds support for simulating the Black Basta attack.

  - [Virtual Network Sensor version 1.0.1495](rn-vns-101495.mdx)

    Includes updates and adds support for simulating the APT28 attack.

  - [Virtual Network Sensor version 1.0.1479](rn-vns-1479.mdx)

    Includes updates and adds support for QUIC protocol. View the full article for more information.

- March 2025

  - [Virtual Network Sensor version 1.0.1449](rn-vns-101449.mdx)

    Includes enhancements, bug fixes, and security updates.

  - [Virtual Network Sensor version 1.0.1448](rn-vns-101448.mdx)

    Includes updates and enhances network monitoring and threat detection capabilities.

- January 2025

  - [Virtual Network Sensor version 1.0.1419](virtual-network-sensor-101419.mdx)

    Includes enhancements, bug fixes, and security updates.

For releases older than six months, use the following links to review release notes.
