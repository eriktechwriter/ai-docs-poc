# Turn off scanning in AWS buckets

Stop File Security Storage from scanning Amazon Web Services (AWS) buckets

To stop File Security from scanning AWS buckets, turn off EventBridge.

### Procedure {#procedure}

1.  On the **Inventory** tab, select the buckets that you want to stop scanning.

2.  Click **Turn Off EventBridge**.

3.  Verify the selected buckets.

4.  Click **Turn Off**.

    The Status changes to EventBridge off and the indicator circle turns black.
:::
