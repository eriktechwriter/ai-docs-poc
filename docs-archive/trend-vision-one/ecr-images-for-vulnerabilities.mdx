# Assess language packages in ECR images for vulnerabilities

April 22, 2024—the Vulnerability Assessment service available in **Cyber Risk Exposure Management** now supports scanning language packages used in your ECR container images. For information on supported languages, see [Vulnerability Assessment supported language packages](vulnerability-assess-languages.mdx).

**Cyber Risk Exposure Management → Threat and Exposure Management**
