# Supported files for Sandbox Analysis

Understand the files that Internet Access submits to the Sandbox Analysis app for analysis.

- Highly suspicious files

  Highly suspicious files are suspected to be fraudulent or possible sources of threats as detected by the Advanced Threat Scan Engine (ATSE). The Advanced Threat Scan Engine (ATSE) uses a combination of pattern-based scanning and heuristic scanning to detect document exploits and other threats used in targeted attacks.

- Portable executables, and Office and PDF files
