# Cloud accounts

Any Cloud "accounts" you add to your Cloud Risk Management organization are represented with the name and environment you set during setup.

Cloud Accounts contain all your account's resource checks that confirm your level of compliance with the [rules](introduction-rules.mdx) run against your Cloud account.
