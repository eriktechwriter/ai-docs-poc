# Running the email attack scenario

Enable Email Sensor on one testing email account, and then run the simulation test to trigger detections in Trend Vision One.

### Procedure {#procedure}

1.  Enable Email Sensor on an Exchange Online email account.

    1.  Go to **Email and Collaboration Security → Email and Collaboration Sensor**.

    2.  Grant permission to access your Exchange Online mailboxes on the **Exchange Online** tab.

    3.  [Enable Email Sensor](managing-email-sensor-detection.mdx) on an email account.

2.  On the Trend Vision One console, click the **Resource Center** icon (![](/images/resourceCenter=73b1d431-813b-467c-8098-62f12bb6e2af.webp)).

3.  Click **Simulations**.

4.  Click **Email Attack**.

    The **Email Attack Simulation** window appears.

5.  Create an email with the following information using an external mailbox.

    - Subject: `Email Demo`

    - Body: `hxxps://ca21-3.winshipway.com/mx-01-006`

      :::note

      Replace “xx” in the URL with “tt”.
      :::

6.  Send the email to the testing email account.

7.  After sending the email, go to the Trend Vision One console to view the expected results.

    :::note

    Results might take a few minutes to appear.
    :::
:::
