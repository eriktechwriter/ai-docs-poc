# Exception List Environment Variable Support

The following table lists the environment variables you can use when adding a file or folder path to the list.

<table>
<colgroup>
<col style="width: 20%" />
<col style="width: 40%" />
<col style="width: 40%" />
</colgroup>
<thead>
<tr>
<th><p>Environment Variable</p></th>
<th><p>Example</p></th>
<th><p>Equivalent Path</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>$allappdata$</p></td>
<td><p>$allappdata$\test\sample.exe</p></td>
<td><p>C:\ProgramData\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$allprograms$</p></td>
<td><p>$allprograms$\test\sample.exe</p></td>
<td><p>C:\ProgramData\Microsoft\Windows\Start Menu\Programs\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$programdir$</p></td>
<td><p>$programdir$\test\sample.exe</p></td>
<td><p>C:\Program Files\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$programdirx86$</p></td>
<td><p>$programdirx86$\test\sample.exe</p></td>
<td><p>C:\Program Files (x86)\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$rootdir$</p></td>
<td><p>$rootdir$\test\sample.exe</p></td>
<td><p>C:\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$systemdir$</p></td>
<td><p>$systemdir$\test\sample.exe</p></td>
<td><p>C:\Windows\System32\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$systemdirx86$</p></td>
<td><p>$systemdirx86$\test\sample.exe</p></td>
<td><p>C:\Windows\SysWOW64\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$tempdir$</p></td>
<td><p>$tempdir$\test\sample.exe</p></td>
<td><p>C:\Windows\Temp\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$userprofile$</p></td>
<td><p>$userprofile$\test\sample.exe</p></td>
<td><p>C:\user\{current_user_account}\test\sample.exe</p></td>
</tr>
<tr>
<td><p>$windir$</p></td>
<td><p>$windir$\test\sample.exe</p></td>
<td><p>C:\Windows\test\sample.exe</p></td>
</tr>
</tbody>
</table>
