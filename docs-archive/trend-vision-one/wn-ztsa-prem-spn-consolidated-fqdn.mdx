# Zero Trust Secure Access on-premises gateway supports consolidated FQDN for Smart Protection Network services

November 18, 2024—The Zero Trust Secure Access - Internet Access On-premises Gateway now integrates the Smart Protection Network Proxy for SPN-related service connections. This reduces the number of FQDN items required for firewall exceptions. For more details refer to [Firewall exception requirements for Trend Vision One](firewall-exception-requirements-for.mdx) .

**Zero Trust Secure Access → Secure Access Configuration → Internet Access and AI Service Access Configuration**
