# Security Awareness firewall exceptions

Add FQDNs used by Security Awareness to your firewall allow lists to allow access to Security Awareness content.

Your network firewalls may block FQDNs used to access and deliver training campaign and phishing simulation content. Add the following regional URLs to your firewall exceptions to ensure access to Security Awareness content.
