# InterScan Web Security Features and Settings Migration

Gain a better understanding of which features and settings migrate when you upgrade from InterScan Web Security products, including InterScan Web Security Virtual Appliance and InterScan Web Security Suite, to Zero Trust Secure Access Internet Access.

The following tables detail how InterScan Web Security features and settings migrate when you upgrade to Internet Access.

**Features**

<table>
<colgroup>
<col style="width: 33%" />
<col style="width: 17%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr>
<th><p>Feature</p></th>
<th><p>Migratable</p></th>
<th><p>Notes</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Dashboard</p></td>
<td><p>No</p></td>
<td><p>All relevant information and settings are found in the Zero Trust Secure Access app in Trend Vision One.</p>
<p>Go to <strong>Zero Trust Secure Access → Secure Access Overview</strong> and click the <strong>Internet Access</strong> tab for an overview of the Internet Access service.</p>
<p>Go to <strong>Zero Trust Secure Access → Secure Access Configuration → Internet Access Configuration</strong> to configure your Internet Access settings.</p></td>
</tr>
<tr>
<td><p>Logs</p></td>
<td><p>No</p></td>
<td><p>Activity logs are available via <strong>XDR Data Explorer</strong> in Trend Vision One.</p>
<p>Go to <strong>Agentic SIEM &amp; XDR → XDR Data Explorer</strong> and select <strong>Secure Access Activity Data</strong> in the <strong>Search Method</strong> dropdown menu.</p>


:::warning[Important]

<p>Your existing logs in InterScan Web Security are not migrated to Trend Vision One.</p>


:::

</td>
</tr>
<tr>
<td><p>Reports</p></td>
<td><p>No</p></td>
<td><p>Reports are available via the Reports app in Trend Vision One.</p>
<p>Go to <strong>Dashboards and Reports → Reports</strong>.</p>


:::warning[Important]

<p>Your existing reports in InterScan Web Security are not migrated to Trend Vision One.</p>


:::

</td>
</tr>
<tr>
<td><p>Updates</p></td>
<td><p>No</p></td>
<td><p>Zero Trust Secure Access Internet Access pattern and engine updates occur every 8 hours from midnight local time. The schedule is not configurable.</p></td>
</tr>
<tr>
<td><p>Notifications</p></td>
<td><p>No</p></td>
<td><p>Notifications are not migrated to Trend Vision One.</p>
<p>Go to <strong>Zero Trust Secure Access → Secure Access Configuration → Customization Settings</strong> and select the related block/warning page to customize your notifications.</p></td>
</tr>
<tr>
<td><p>Audit Log</p></td>
<td><p>No</p></td>
<td><p>Log auditing is available via the <strong>Audit Logs</strong> app.</p>
<p>Go to <strong>Administration → Audit Logs</strong> and select <strong>Zero Trust Secure Access</strong> from the <strong>Category</strong> dropdown menu.</p></td>
</tr>
<tr>
<td><p>Deployment Wizard</p></td>
<td><p>No</p></td>
<td><p>The Zero Trust Secure Access app features a deployment wizard to assist you in setting up various services.</p>
<p>Go to <strong>Zero Trust Secure Access → Secure Access Overview</strong> and click <strong>Deployment Wizard</strong>. In the <strong>Internet Access</strong> section, click <strong>Configure</strong> to launch the wizard.</p></td>
</tr>
</tbody>
</table>
