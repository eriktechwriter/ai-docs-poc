# Advanced Search Categories

During an Advanced Search, use the following search criteria options for **Users** and **Endpoints**.

**User Categories**

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Category</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>User name</p></td>
<td><p>The account name of local users or people belonging to an Active Directory structure</p></td>
</tr>
<tr>
<td><p>Direct manager</p></td>
<td><p>The account name of the person who users are assigned to report to</p></td>
</tr>
<tr>
<td><p>Location in Active Directory</p></td>
<td><p>The organization unit from which to begin your search</p></td>
</tr>
<tr>
<td><p>Department</p></td>
<td><p>The name of the department in your company that groups users based on their function (for example, Accounting) or other criteria</p></td>
</tr>
<tr>
<td><p>Active Directory group</p></td>
<td><p>A collection of Active Directory user and computer accounts, contacts and other groups that can be managed as a single unit</p></td>
</tr>
<tr>
<td><p>Threat type</p></td>
<td><p>Select a security threat type from the third drop-down list</p></td>
</tr>
<tr>
<td><p>Security threat</p></td>
<td><p>Search for a specific security threat by typing a malware name, URL, IP address, or sender email address</p></td>
</tr>
<tr>
<td><p>Threat status</p></td>
<td><p>The remediation status indicated by the flag icon in the first column on the <strong>Security Threats</strong> screen: Resolved by product, Action required, Resolved manually</p>
<p>For more information, see <a href="trend-vision-one-security-threats-on-users">Security Threats for Users</a>.</p></td>
</tr>
<tr>
<td><p>Importance</p></td>
<td><p>The assigned importance level</p>


:::note

<p>You cannot save advanced search filters that contain important users/endpoints.</p>


:::


<p>For more information, see <a href="trend-vision-one-user-or-endpoint-importance">User or Endpoint Importance</a>.</p></td>
</tr>
<tr>
<td><p>Active Directory site</p></td>
<td><p>The site name synchronized from the Active Directory</p>
<p>For more information, see <a href="trend-vision-one-endpoint-user-grouping">Endpoint and User Grouping</a>.</p></td>
</tr>
<tr>
<td><p>Reporting line</p></td>
<td><p>The display name for the reporting line synchronized from the Active Directory</p>
<p>For more information, see <a href="trend-vision-one-endpoint-user-grouping">Endpoint and User Grouping</a>.</p></td>
</tr>
</tbody>
</table>
