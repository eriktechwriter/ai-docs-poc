# System requirements

::**Mobile Security** is compatible with Android, iOS/iPadOS, and ChromeOS operating systems. Find system requirements for each operating system below.

:::note

ChromeOS has no minimum system requirements because Chromebooks run an Android subsystem and update to the latest version automatically.
:::
