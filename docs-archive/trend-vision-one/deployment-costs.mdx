To track the costs of your Agentless Vulnerability & Threat Detection scanner stack:

1.  Go to the AWS Cost Management console and sign in.

2.  [Enable the AWS Cost Explorer feature](https://docs.aws.amazon.com/cost-management/latest/userguide/ce-enable.mdx).

3.  [Activate the AppManagerCFNStackKey cost allocation tag.](https://docs.aws.amazon.com/solutions/latest/network-orchestration-aws-transit-gateway/activate-cost-allocation-tags-associated-with-the-solution.mdx)

After you have enabled AWS Cost Explorer, you can also track the costs of Agentless Vulnerability & Threat Detection EBS operations.

1.  Go to the AWS Cost Explorer dashboard and sign in.

2.  Set the **Group by** dimension to **API operation**.

3.  Configure the following filters:
