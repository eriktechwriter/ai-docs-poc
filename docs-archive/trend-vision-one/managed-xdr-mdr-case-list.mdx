# MDR (Managed XDR) case list

Interact with the Trend Micro Managed Services team from within **Case Management**. You can get incident alerts and recommended remediation actions, as well as provide additional case information.

The **MDR** tab in **Case Management** displays the cases that the Trend Micro Managed Services team has opened for you.

:::warning[Important]

The **MDR** tab is only available for [Managed Services](managed-services.mdx) customers.
:::
