# Configuring phishing criteria

### Procedure {#procedure}

1.  Select "**Phishing and other suspicious content**".

    :::note

    Cloud Email Gateway Protection leverages Trend Micro Antispam Engine to filter email messages for spam and phishing incidents. Email messages will be categorized as phishing threats if Trend Micro Antispam Engine detects phishing and other suspicious content in those messages.
    :::
:::
