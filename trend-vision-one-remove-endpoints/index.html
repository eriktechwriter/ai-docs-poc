Remove Endpoints

Learn how to remove endpoints from your Endpoint Inventory.

You can remove endpoints directly from the Endpoint Inventory screen.

Important

Removing an endpoint does not uninstall the Trend Vision One agent from the endpoint or stop the agent program sending information to Trend Vision One. For information on how to uninstall the agent from the endpoint, see Uninstalling Agents.

Endpoint Inventory currently only supports removal of sensor only endpoints.
