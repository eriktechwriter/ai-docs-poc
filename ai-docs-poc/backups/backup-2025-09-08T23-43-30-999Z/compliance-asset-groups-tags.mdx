Learn how to create asset groups and assign asset tags to your endpoint assets.

### Procedure {#procedure}

1.  Go to **Service Management → Asset Group Management**.

2.  On the **Asset Group Management** screen, click **Get started** to create your first asset group.

3.  Hover over the organization node and click the ![](/images/add_icon=cf892c2f-1a1f-4d22-848f-023067e4a507.webp) icon to add an asset group.

    :::note

    For more information, see [Create an asset group](creating-asset-group.mdx).
    :::

4.  Type a name and assign an asset group tag value to the assets in the group.

5.  Assign an asset group tag value or select an existing value from the drop-down list.

    :::note

    Tag values must be unique. You can only select tag values that are not already in use in another asset group.
    :::

6.  Type a **Description** of the asset group.

7.  Click **Save**.

8.  Go to **Cyber Risk Exposure Management → Attack Surface Discovery**.

9.  Open the **Manage Tags** screen.

    :::tip

    The following actions can be used to open the **Manage Tags** screen:

    - Select one or more assets from the asset list and click **Manage Tags**.

    - Hover over the name of an asset on the asset profile screen, select the **Custom Tags** tab, and click **Manage Tags**.

    - Go to a specific asset profile and click **Manage Tags** in the sidebar.

    - Go to **Endpoint Security → Endpoint Inventory**, select any of the available endpoints and click **Manage tags**.
    :::

10. On the **Manage Tags** screen in the **Available tags** list, select any of the available asset group tag values that you assigned to your groups.

11. Click the right arrow to move the tags to **Tags assigned to all selected assets**.

12. Click **Save**.

    :::note

    For more information, see [Custom asset tags](custom-asset-tags.mdx).
    :::

13. Go to **Cyber Risk Exposure Management → Compliance Management**.

14. On the **Overview** tab, click **View asset group pass rates**.

    The **Asset Group Pass Rates by Framework or Standard** window appears.
:::
