Create, edit, and delete your automated tagging rules.

The following table outlines the actions available on the **Automated tagging** screen.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Action</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Create automated tagging rules</p></td>
<td><p>Click <strong>+ Create rule</strong> to <a href="trend-vision-one-create-tagging-rules">create rules</a> for automated tagging.</p></td>
</tr>
<tr>
<td><p>Edit automated tagging rules</p></td>
<td><p>To modify an existing rule, click the <img src="./images/edit_icon=GUID-1F1D1164-5310-4D6D-ACD0-6049C86960AF.webp" /> icon to open the <strong>Edit a tagging rule</strong> window, make the required changes, then click <strong>Save</strong>.</p>


:::note

<p>You cannot edit the <strong>Asset type</strong> targeted by an automated tagging rule.</p>
<p>When you edit a rule, all tags previously assigned to assets by the rule remain unchanged. Any new tags specified for the rule are assigned to assets the next time the rule runs.</p>


:::

</td>
</tr>
<tr>
<td><p>Run automated tagging rules</p></td>
<td><p>Click the <img src="./images/run_icon=cbe6ecd0-17e8-4e04-bef3-4efe4eb9c7e5.webp" /> icon to immediately run an enabled rule.</p>


:::note

<p>To run a disabled rule, you must first edit the rule and change the status to <strong>Enabled</strong>.</p>


:::

</td>
</tr>
<tr>
<td><p>Delete automated tagging rules</p></td>
<td><p>Click the <img src="./images/trash_icon=GUID-47cf6867-6315-438e-8670-86ff36f22a28.webp" /> icon and click <strong>Delete</strong> in the confirmation window to delete a tagging rule.</p>


:::note

<p>When you delete a rule, all tags previously assigned to assets by the rule remain unchanged. To remove those tags, go to <strong>Tag library</strong> and manually remove the tags from the assigned assets</p>


:::

</td>
</tr>
</tbody>
</table>
