You can disable the scanning in multiple mount points at the same time.

### Procedure {#procedure}

1.  In Trend Vision One go to **Cloud Security â†’ File Security**.

2.  Click **Inventory**.

3.  Click the **Virtual Appliance** node.

4.  Select the checkboxes of the **Folder Paths** whose scanning you want to enable.

5.  From the **Manage scanning configuration** list, select **Disable scanning**.

6.  Click **Disable**.
:::
