:::: {role="main"}
Import or export lists of **Device Control Allow Devices** that apply to all Apex One Security Agent policy targets.

:::note

- Only Trend Vision One Endpoint Security agents with Data Protection enabled override the “Block” or “Read” action on devices added to the Device Control Allowed Devices list.

- The Device Control Allowed Devices list does not apply to Trend Vision One Endpoint Security agents without Data Protection and Trend Vision One Endpoint Security agents with Device Control permission not set to “Block” or “Read”.
:::
