### Add-in for Outlook Version 0.1.162 {#add-in-for-outlook-version-0.1.162}

Latest version

This version includes the following features:

- Allow your users to view and take action on their quarantined emails.

- Enables your users to report Exchange Online emails to Trend Micro as false positives and false negatives to help improve threat detection.
