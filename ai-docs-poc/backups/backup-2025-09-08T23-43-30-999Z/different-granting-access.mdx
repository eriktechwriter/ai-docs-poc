::: {role="main"}
There are different ways to begin granting Cloud Email and Collaboration Protection access to services:

- Go to **Email and Collaboration Security Operations → Cloud Email and Collaboration Protection → Dashboard → Service Status**, click **Grant Access** in the **Action** column for a service.

- Go to **Email and Collaboration Security Operations → Cloud Email and Collaboration Protection → Administration → Service Account**, click **Add**, hover over the organization for which you need to grant access to services, and select a service from the list that appears.
:::
