:::: {role="main"}
Allow the necessary mobile platform permissions to ensure all **Mobile Security** features function properly. Note that some applications, such as banking applications may not function correctly after enabling the VPN or Accessibility. For more information, see [VPN and Accessibility Service compatibility](vpn-access-service-compatibility.mdx).

The following tables outline the permissions that you must allow for Android and iOS/iPadOS devices.

:::note

Users must manually grant permissions to permissions and services not managed by your MDM solution.
:::
