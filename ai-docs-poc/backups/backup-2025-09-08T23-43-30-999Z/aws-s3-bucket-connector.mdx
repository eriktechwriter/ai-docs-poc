Share XDR data with your AWS environment by configuring the AWS S3 bucket connector.

The AWS S3 bucket connector utilizes AWS cross account trust relationships to send XDR data to a configured S3 bucket. You can connect multiple S3 buckets and configure the data sent to each bucket individually. The AWS S3 bucket connector requires a [custom AWS IAM role](roles-aws-s3-bucket.mdx) with custom permissions to connect an S3 bucket.
