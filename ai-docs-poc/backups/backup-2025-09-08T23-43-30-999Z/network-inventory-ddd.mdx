Manage Deep Discovery Inspector appliances using a connected Deep Discovery Director appliance.

Network Inventory allows you to connect your Deep Discovery Director, automatically connecting all managed Deep Discovery Inspector appliances. You can monitor the connection status of each Deep Discovery Inspector appliance, as well as access the consoles.

The following table outlines actions available on the **Network Inventory** screen.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Action</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Connect a Deep Discovery Director appliance</p></td>
<td><p>Click <strong>Connect Appliance</strong> to connect a Deep Discovery Director appliance.</p>
<p>Connecting a Deep Discovery Director appliance automatically connects all managed Deep Discovery Inspector appliances. For more information, see <a href="trend-vision-one-connecting-deep-discovery-director">Connecting through Deep Discovery Director</a>.</p></td>
</tr>
<tr>
<td><p>View and monitor bandwidth usage</p></td>
<td><p>View the current allocated and total bandwidth usage, and click <strong>Bandwidth usage</strong> to view bandwidth usage for all connected appliances, as well as the allocated and available bandwidth for your Trend Micro license.</p></td>
</tr>
<tr>
<td><p>View connected appliances</p></td>
<td><p>Click the <strong>Connected Network Sensor Appliances</strong> to view connected Deep Discovery Inspector appliances.</p>
<p>To access the connected Deep Discovery Inspector console, click <strong>Access console</strong>.</p>
<p>To configure the Network Sensor, see <a href="trend-vision-one-configuring-network-sensors-ddd">Configuring Network Sensors with Deep Discovery Director</a></p></td>
</tr>
<tr>
<td><p>View unsupported appliances</p></td>
<td><p>Click the <strong>Unsupported Network Sensor Appliances</strong> to view a list of unsupported Deep Discovery Inspector appliances managed by your Deep Discovery Director. For full support, update to the latest version of Deep Discovery Inspector.</p></td>
</tr>
<tr>
<td><p>Access the Deep Discovery Director console</p></td>
<td><p>Click the console icon (<img src="./images/Network_Inventory_DD_Console=GUID-DD9ECC14-E9FA-41ED-BB02-95F5264FA946=1=en-us=Low.webp" />) to open the Deep Discovery Director console in a new browser window or tab.</p></td>
</tr>
<tr>
<td><p>Reset Network Inventory</p></td>
<td><p>Click <img src="./images/vertical_ellipsis_icon=GUID-20230329141212.webp" /> and then click <strong>Reset Network Inventory</strong> to disconnect all appliances from Network Inventory.</p>


:::warning[Important]

<p>Resetting Network Inventory disconnects all appliances. After resetting, Workbench is unable to generate Network Analytics reports from previously collected data. Use this feature only if you are changing how your appliances connect to Network Inventory.</p>


:::

</td>
</tr>
</tbody>
</table>
