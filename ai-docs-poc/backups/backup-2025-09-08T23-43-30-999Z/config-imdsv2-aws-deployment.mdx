::: {role="main"}
Trend Micro supports IMDSv2 in AWS Private Access connectors by using Launch Templates. By modifying the Launch Template and redeploying the connectors, you can enable the use of IMDSv2.

If Private Access Connectors are already deployed, the connector EC2s can be switched to IMDSv2 without downtime.
