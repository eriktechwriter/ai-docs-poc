::: {role="main"}
In Trend Micro Apex Central or Control Manager, access the **Apex One (Mac) Key Performance Indicators** widget on the **Dashboard** to perform the following indicator-related tasks.

**KPI Widget Indicator Tasks**

<table>
<colgroup>
<col style="width: 38%" />
<col style="width: 62%" />
</colgroup>
<thead>
<tr>
<th><p>Task</p></th>
<th><p>Steps</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Add a new indicator</p></td>
<td><ol>
<li><p>Click <strong>Add Indicator</strong>. The <strong>Add Indicator</strong> screen appears.</p></li>
<li><p>Select an option from the <strong>Name</strong> drop-down list and optionally customize settings.</p></li>
<li><p>Click <strong>Save</strong>.</p></li>
</ol></td>
</tr>
<tr>
<td><p>Edit an indicator</p></td>
<td><ol>
<li><p>Click the indicator in the list. The <strong>Edit Indicator</strong> screen appears.</p></li>
<li><p>Customize settings.</p></li>
<li><p>Click <strong>Save</strong>.</p></li>
</ol></td>
</tr>
<tr>
<td><p>Delete an indicator</p></td>
<td><ol>
<li><p>Click the indicator in the list. The <strong>Edit Indicator</strong> screen appears.</p></li>
<li><p>Click <strong>Delete</strong>.</p></li>
<li><p>Click <strong>OK</strong>.</p></li>
</ol></td>
</tr>
<tr>
<td><p>Configure event threshold settings</p></td>
<td><ol>
<li><p>On the <strong>Add Indicator</strong> or <strong>Edit Indicator</strong> screen, select <strong>Enable alerts at the following thresholds</strong>.</p></li>
<li><p>Type the minimum number of event occurrences for each event type.</p></li>
<li><p>Click <strong>Save</strong>.</p></li>
</ol>


:::note

<p>The important or critical icon displays in the <strong>Occurrences</strong> column if both of the following are true:</p>
<ul>
<li><p>The number of event occurrences that match this indicator is equal to or more than the threshold.</p></li>
<li><p><strong>Enable alerts at the following threshold</strong> is selected.</p></li>
</ul>


:::

</td>
</tr>
</tbody>
</table>
:::
