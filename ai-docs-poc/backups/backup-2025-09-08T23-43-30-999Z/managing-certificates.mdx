Manage server certificates for internal applications and enrollment certificates for Zero Trust Secure Access modules and connectors.

On the certificate management screen, you can manage certificates on the following tabs.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Tab</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Server Certificates</p></td>
<td><p>Add a server certificate to faciliate Browser Access to internal applications, or you can use the default self-signed certificate provided by Trend Vision One.</p>
<p>For more information, see <a href="trend-vision-one-adding-server-certificate">Adding a server certificate</a>.</p></td>
</tr>
<tr>
<td><p>Enrollment Certificates</p></td>
<td><p>Add an enrollment certificate to generate client certificates for Zero Trust Secure Access modules and connectors, or you can use the default certificate provided by Trend Vision One.</p>
<p>Click the <strong>Enabled</strong> radio button to assign the selected certificates to your organization's modules and connectors.</p>
<p>For more information, see <a href="trend-vision-one-adding-enrollment-certificate">Adding an enrollment certificate</a>.</p></td>
</tr>
</tbody>
</table>

- [Adding an enrollment certificate](adding-enrollment-certificate.md "Add an enrollment certificate to generate client certificates for Zero Trust Secure Access modules and connectors.")
