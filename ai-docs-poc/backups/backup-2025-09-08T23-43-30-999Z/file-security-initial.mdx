April 8, 2024—Trend Vision One File Security is a scanning service that can detect all types of malicious software (malware) including trojans, ransomware, spyware, and more.

File Security offers the following features and enhancements:

<td><p>File Security Storage provides anti-malware scanning on files in cloud storage services such as Amazon Web Services (AWS). This means you can integrate automated scanning into your continuous integration and continuous delivery and deployment (CI/CD) pipeline. Then effortlessly detect all types of malware including viruses, trojans, spyware, and more.</p>
<p>For more information, see <a href="trend-vision-one-cloud-storage-scanning">File Security Storage</a>.</p></td>
