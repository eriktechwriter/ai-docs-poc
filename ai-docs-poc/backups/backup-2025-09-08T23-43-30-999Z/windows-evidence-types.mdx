View the types of data and information supported for collection from Windows endpoints.

The following categories describe the types of evidence the [Incident Response Evidence Collection playbooks](create-ir-collection-playbooks.mdx), [Collect Evidence response task](collect-evidence-task.mdx), and [Trend Micro Incident Response Toolkit](manual-evidence-collection.mdx) collect from Windows endpoints.

These evidence types appear in columns after selecting an evidence category when examining an Evidence Report.
