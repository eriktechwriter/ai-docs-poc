View attributed and descriptions within the shared file info object category collected from Linux endpoints.

The following table contains descriptions of the attributes in the shared file info objects category that the [Collect Evidence task](collect-evidence-task.mdx) and [Trend Micro Incident Response Toolkit](manual-evidence-collection.mdx) may collect from Linux endpoints. These attributes are displayed associated with other evidence types when examining an [Evidence Report](evidence-report.mdx).

:::note

Shared file info objects may appear in multiple evidence categories or types.
