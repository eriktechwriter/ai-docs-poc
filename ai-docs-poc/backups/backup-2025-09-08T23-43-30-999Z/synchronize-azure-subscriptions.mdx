Server & Workload Protection automatically synchronizes with your connected Azure subscriptions every 10 minutes. You can manually synchronize an Azure subscription to obtain and display the latest set of Azure virtual machines.

### Procedure {#procedure}

1.  In the Trend Vision One console, go to **Endpoint Security â†’ Server & Workload Protection**.

2.  In the Server & Workload Protection manager, go to **Computers**.

3.  Locate the Azure subscription you want to synchronize in the folder list.

4.  Right-click the subscription and select **Synchronize Now**.
:::
