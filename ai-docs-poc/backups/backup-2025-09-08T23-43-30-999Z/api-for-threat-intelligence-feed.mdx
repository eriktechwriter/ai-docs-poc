Integrate the Trend Threat Intelligence Feed into your security environment using the application programming interface (API) to receive the latest, continuously updated threat intelligence.

### Procedure {#procedure}

1.  In Trend Vision One, go to **Administration → Credits & Billing**.

2.  Click **Set Up** for Trend Threat Intelligence Feed.

    The **Credit Settings** panel appears.

3.  Enable **Trend Threat Intelligence Feed** or **Trend Threat Intelligence Feed for Service Providers**.

4.  Click **Save**.

5.  Go to **Administration → API Keys**.

6.  Click **Add API Key**.

    The **Add API Key** panel appears.

7.  Specify a descriptive name for the API key.

8.  Select a role from the drop-down list.

9.  Select an expiration time from the drop-down list.

10. Click **Add**.

11. Copy and save the API key value.

12. Add the Trend Threat Intelligence Feed API key to your own security environment.

    :::note

    For more information about API query parameters, see the [Automation Center](https://automation.trendmicro.com/xdr/api-v3).
    :::
:::
