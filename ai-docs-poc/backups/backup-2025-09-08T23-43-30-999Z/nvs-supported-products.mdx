Learn more about the products supported for vulnerability scanning by the Network Vulnerability Scanner service.

Network Vulnerability Scanner supports the following products running supported operating system versions. When authentication credentials are required for the scan template, the account associated with the credentials must have the following permissions. Scans for each product run the following commands to obtain device information.

To enable scanning for a particular product, ensure the Network Vulnerability Scanner service on your Service Gateway is updated to at least the required version.
