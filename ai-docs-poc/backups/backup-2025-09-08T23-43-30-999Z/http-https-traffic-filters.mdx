:::: {role="main"}
Manage HTTP/HTTPS traffic filters used in internet access control rules to monitor and manage internet access based on request methods, URLs, and header fields of HTTP and HTTPS request or response traffic.

:::note

HTTP/HTTPS traffic filters can be applied to the following:

- Internet access rules
:::
