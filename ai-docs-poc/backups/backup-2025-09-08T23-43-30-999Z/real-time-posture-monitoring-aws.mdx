May 8, 2024—Cloud Risk Management now supports [Real-Time Posture Monitoring](real-time-threat-monitoring.mdx) previously titled Real-Time Threat Monitoring (RTM) for AWS accounts connected through the Cloud Accounts app. You can enable Real-Time Posture Monitoring while connecting a new AWS account and organization or turn the feature on for existing AWS accounts or organizations.

**Cyber Risk Exposure Management → Cloud Security Posture → Misconfiguration and Compliance**
:::
