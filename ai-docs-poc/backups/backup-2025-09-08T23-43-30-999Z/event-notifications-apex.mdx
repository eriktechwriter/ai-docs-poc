Trend Micro Apex Central can notify individuals or groups of recipients about events detected by managed products.

Use the **Event Notifications** screen to enable or disable notifications about events from the following categories.

<table>
<colgroup>
<col style="width: 40%" />
<col style="width: 60%" />
</colgroup>
<thead>
<tr>
<th><p>Event Category</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Advanced Threat Activity</p></td>
<td><p>Provides warnings about advanced and unknown threats</p>
<p>For more information, see <a href="trend-vision-one-advanced-threat-activity-events">Advanced Threat Activity Events</a>.</p></td>
</tr>
<tr>
<td><p>Content Policy Violation</p></td>
<td><p>Provides warnings about email content and URL security policy violations</p>
<p>For more information, see <a href="trend-vision-one-content-policy-violation-events">Content Policy Violation Events</a>.</p></td>
</tr>
<tr>
<td><p>Data Loss Prevention</p></td>
<td><p>Provides information about Data Loss Prevention incidents and template matches</p>
<p>For more information, see <a href="trend-vision-one-data-loss-prevention-events">Data Loss Prevention Events</a>.</p></td>
</tr>
<tr>
<td><p>Known Threat Activity</p></td>
<td><p>Provides warnings about viruses/spyware/grayware detected by antivirus managed products</p>
<p>For more information, see <a href="trend-vision-one-known-threat-activity-events">Known Threat Activity Events</a>.</p></td>
</tr>
<tr>
<td><p>Network Access Control</p></td>
<td><p>Provides warnings from managed Network VirusWall products</p>
<p>For more information, see <a href="trend-vision-one-network-access-control-events">Network Access Control Events</a>.</p></td>
</tr>
<tr>
<td><p>Unusual Product Behavior</p></td>
<td><p>Provides information about product options or service activation and deactivation</p>
<p>For more information, see <a href="trend-vision-one-unusual-product-behavior-events">Unusual Product Behavior Events</a>.</p></td>
</tr>
<tr>
<td><p>Updates</p></td>
<td><p>Provides antivirus and content security component update results (successful or unsuccessful)</p>
<p>For more information, see <a href="trend-vision-one-updates-event-notify">Updates</a>.</p></td>
</tr>
