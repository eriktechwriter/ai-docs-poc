June 16, 2025—Trend Vision One now supports a new version of Helm chart for Container Security. This upgrade version improves performance and adds access to new functionality. The Helm chart repository is at a new location in GitHub: <https://github.com/trendmicro/visionone-container-security-helm>.

:::warning[Important]

You must modify your Helm override file values and firewall exceptions for this new version.
:::

For more information, see [Upgrade Helm chart from Trend Cloud One to Trend Vision One](upgrade-helm-cloud-one.mdx).

**Cloud Security → Container Security → Container Inventory**
