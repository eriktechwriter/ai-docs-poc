June 10, 2025—CREM Cloud Risk Management now supports Real-Time Posture Monitoring previously titled Real-Time Monitoring (RTM) for Azure subscriptions connected through the Cloud Accounts app. You can enable Real-Time Posture Monitoring while creating a new Azure subscription or turn the feature on for existing subscription or organizations. For more information, see [Real-Time Posture Monitoring settings](realtime-threat-monitor.mdx)

**Cloud Security → Cloud Security Posture**
