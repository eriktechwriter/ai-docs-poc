April 14, 2025—Workbench insights now have a insight-level network analytics report.

The network analytics report shows network correlations between the objects included in a Workbench insight. The report provides comprehensive analytics to help you monitor, manage, and identify the abnormal network traffic in your environment. [Learn more](network-analytics-report.mdx).
