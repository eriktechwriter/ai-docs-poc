The Trend Vision One Endpoint Security agent can also protect against the addition of processes in the Microsoft Software Restriction Policies (SRP). Software Restriction Policies prevent the listed applications from running on the endpoint. To prevent the addition of Trend Vision One Endpoint Security agent processes in the Software Restriction Policies list:

1.  Enable **Protect Security Agent processes**.

2.  Enable the **Unauthorized Change Prevention Service**.
