:::: {role="main"}
Enable WebGL to achieve high-performance rendering for some Workbench capabilities.

:::warning[Important]

The Trend Vision One platform supports the latest versions of the following desktop browsers:

- Google Chrome

- Mozilla Firefox

- Microsoft Edge (Chromium)

The Trend Vision One platform is not officially supported on mobile browsers (Android and iOS).
:::

### Procedure {#procedure}

1.  Make sure your system has the latest graphics drivers.

    :::note

    For more information, consult the manufacturer's documentation.
    :::

2.  Enable WebGL in your web browser.

    :::warning[Important]

    The following instructions were valid as of April 2025. For further help, check your web browser documentation.
    :::

    - Google Chrome™

      1.  Open Chrome.

      2.  Go to **→ Settings → System**.

      3.  Turn on **Use hardware acceleration when available**.

      4.  Restart Chrome.

    - Firefox

      1.  Open Firefox.

      2.  Go to **→ Settings → General → Performance**.

      3.  Clear the **Use recommended performance settings** checkbox.

      4.  Select **Use hardware acceleration when available**.

      5.  Restart Firefox

    - Microsoft Edge (Chromium)

      1.  Open Edge.

      2.  Go to **→ Settings → System and performance**.

      3.  Turn on **Use hardware acceleration when available**.

      4.  Restart Edge.

3.  Verify WebGL is active by going to <https://get.webgl.org/>.

    A spinning cube animation indicates that WebGL is active.
