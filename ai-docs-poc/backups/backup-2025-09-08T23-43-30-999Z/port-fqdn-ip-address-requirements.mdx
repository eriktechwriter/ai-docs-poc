Configure your firewall exceptions to allow ports and FQDNs required by Zero Trust Secure Access services.

Zero Trust Secure Access requires network access to communicate with your corporate applications, third-party applications, and internet gateways. You must add the required fully qualified domain names (FQDNs) or IP addresses to your firewall rules. In addition to the FQDNs and IP addresses, Zero Trust Secure Access Private Access customers need to open specific ports for the Private Access Connector. Refer to [Firewall exception requirements for Trend Vision One](firewall-exception-requirements-for.mdx) for specific FQDN (or IP address) and port requirements.
