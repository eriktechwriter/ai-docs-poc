November 18, 2024—You can now add custom tags to Trend Vision One resources deployed to your AWS accounts using the Cloud Accounts app. Use custom tags help support cost tracking, automation, and other downstream workflows that rely on AWS tags. The feature supports tagging resources deployed using CloudFormation.

**Cloud Security → Cloud Accounts**
:::
