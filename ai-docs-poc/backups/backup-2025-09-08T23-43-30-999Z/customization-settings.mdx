Customize the content of the Zero Trust Secure Access pages and Secure Access Module notifications shown to users.

### Procedure {#procedure}

1.  On the Trend Vision One console, go to **Zero Trust Secure Access → Secure Access Configuration → Customization Settings**.

    A list of the available pages appears.

2.  Choose the page you want to modify.

3.  Click **Customize** at the bottom of the screen.

    The customizable elements of the page and the **Reset to default** button appear on screen.

4.  Locate and click the element you want to modify.

    A panel with all related customizable fields appears.

5.  Make the desired changes in the available fields.

    - Text fields: Some fields support the use of tokens. For a complete list of the available tokens, refer to the list on the UI.

    - Images: Image size must not exceed 100 KB. Recommended dimensions for page icon are 24 x 24 pixels. Supported file types: JPEG, JPG, PNG, SVG.

6.  Click **Preview**.

    A preview of the changes appears.

7.  (Optional) Revert all applied customizations on the selected page by clicking **Reset to default**.

    A preview of the changes appears.

8.  Click **Save** to save your changes.

    :::warning[Important]

    You must save the changes displayed in the preview before the customizations apply to the actual console.
    :::
:::
