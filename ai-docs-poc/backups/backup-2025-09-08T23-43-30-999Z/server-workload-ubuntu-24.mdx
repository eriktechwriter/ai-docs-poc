September 30, 2024—The Trend Endpoint Agent now supports deploying Server & Workload Protection and Endpoint Sensor to Ubuntu 24.04 64-bit (x86-64) endpoints. You can select the Ubuntu 24.04 distribution when deploying a new agent in the Endpoint Inventory app. For details on supported Linux platforms, see [Trend Vision One Endpoint Security agent system requirements](endpoint-agent-system-requirements.mdx).

**Endpoint Security → Endpoint Inventory**
