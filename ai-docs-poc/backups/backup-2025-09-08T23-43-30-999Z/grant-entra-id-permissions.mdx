:::: {role="main"}
Grant permissions from your Microsoft Entra ID tenants for use in Identity Inventory to better see and manage the identities in your organization.

**Identity Inventory** supports Microsoft Entra ID accounts with one or more tenants.

:::warning[Important]

Sign-in data is unavailable for Microsoft Entra ID accounts with free licenses. If you have a free license, some data fields may be incomplete after syncing.
:::

### Procedure {#procedure}

1.  Grant read permissions in your Microsoft Entra ID tenant to Trend Vision One to allow data sync with Identity Inventory.

    1.  Go to **Workflow and Automation → Third-Party Integration** and select **Microsoft Entra ID** from the list of integrations.

    2.  Grant the following permissions: *Read user and device information, cloud app data, and activity data.* For more information, see [Configure Microsoft Entra ID integration](configuring-integration.mdx).

2.  Grant write permissions in your Microsoft Entra ID tenant to Trend Vision One to make enforcement actions available in Identity Inventory.

    1.  In **Workflow and Automation → Third-Party Integration**, select **Microsoft Entra ID** from the list of integrations.

    2.  Grant the following permissions: *Read directory data and perform account management actions.* For more information, see [Configure Microsoft Entra ID integration](configuring-integration.mdx).
::::
