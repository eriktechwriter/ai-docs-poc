May 29, 2025—You can now conduct manual scans for vulnerabilities and malware on connected cloud accounts that have Agentless Vulnerability & Threat Detection enabled. To trigger a manual scan, go to Cloud Security Posture and click **Scan** next to the Agentless Vulnerability & Threat Detection entry in the **Available features** widget. To learn more, see [Scan manually for vulnerabilities and malware](avtd-manual-scan.mdx).

**Cloud Security → Cloud Security Posture → Cloud Security Posture**
:::
