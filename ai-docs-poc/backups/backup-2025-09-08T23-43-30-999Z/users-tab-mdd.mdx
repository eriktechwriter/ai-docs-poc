View information about users and manage users on the **Users** tab.

You can view the following information on the **Users** tab.

<table>
<colgroup>
<col style="width: 33%" />
<col style="width: 67%" />
</colgroup>
<thead>
<tr>
<th><p>Data</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>User</p></td>
<td><p>The user's name</p></td>
</tr>
<tr>
<td><p>Email address</p></td>
<td><p>The user's email address</p></td>
</tr>
<tr>
<td><p>Device count</p></td>
<td><p>The total number of enrolled devices owned by the user</p></td>
</tr>
<tr>
<td><p>Assignment group</p></td>
<td><p>The assignment group that the user belongs to</p></td>
</tr>
</tbody>
</table>

You can perform the following actions on the **Users** tab.

<table>
<colgroup>
<col style="width: 33%" />
<col style="width: 67%" />
</colgroup>
<thead>
<tr>
<th><p>Action</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Filter user data</p></td>
<td><p>Use the search box to locate specific user data.</p></td>
</tr>
<tr>
<td><p>Delete users</p></td>
<td><p>Click the Delete action icon (<img src="./images/delete_policy=cce60f96-0e5c-4565-895f-d648c42e8495.webp" />) corresponding to a user, or select one or more users from the list and click <strong>Delete</strong>.</p>
<p>Deleting a user also deletes and unenrolls any devices associated with the user.</p></td>
</tr>
<tr>
<td><p>Show the devices owned by a user</p></td>
<td><p>In the <strong>Device count</strong> column, click the number to see a list of enrolled devices filtered by "User:".</p></td>
</tr>
<tr>
<td><p>Delete users that have already been deleted from Microsoft Entra ID</p></td>
<td><p>Hover over the information icon (<img src="./images/information_icon=f933cb98-a237-4e4b-844e-cfad9894d2a7.webp" />) corresponding to a user, or click <strong>Delete from Mobile Inventory</strong> on the confirmation dialog box that appears when any users are found to be deleted from Microsoft Entra ID.</p></td>
</tr>
</tbody>
</table>
