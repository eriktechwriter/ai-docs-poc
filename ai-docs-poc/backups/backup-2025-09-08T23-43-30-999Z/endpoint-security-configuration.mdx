Configure settings and policies to manage your Trend Vision One Endpoint Security agent deployments.

Endpoint Security Configuration provides an easy to find location to access Trend Vision One Endpoint Security agent settings, configurations, and policies. Follow the links below to learn more about each feature:

- [Endpoint Security Policies](sensor-settings-policies.mdx): create and configure policies to manage settings such as detection and response, aggressive mode, and advanced risk telemetry.

- [Version Control Policies](security-update-policies.mdx): create and configure policies to manage agent update settings, such as version control for both the agent program and detection components.
