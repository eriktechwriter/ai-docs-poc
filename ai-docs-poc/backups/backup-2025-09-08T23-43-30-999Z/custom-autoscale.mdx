Scale your Private Access Connector VM instances based on metric thresholds by creating a custom autoscale policy.

### Procedure {#procedure}

1.  In the left navigation, click **Scaling**.

2.  On the **Configure** tab of the **Scaling** screen that appears, select **Custom autoscale**.

3.  In the **Custom autoscale** configuration area, set **Scale mode** to **Scale based on a metric**.

4.  Add a "Scale out" rule to increase the instance count by clicking **Add a rule** and specifying the following fields.
