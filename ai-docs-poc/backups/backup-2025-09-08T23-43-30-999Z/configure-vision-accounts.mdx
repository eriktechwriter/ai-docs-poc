Create and manage Trend Vision One user accounts to allow authorized users to access the console.

Before creating a SAML account, add your identity provider in the [Identity Providers](identity-providers.mdx) app.

- [Add an IdP-Only SAML Group Account](adding-idp-only-saml-group-account.md "Add multiple users in an assigned group to sign in to the Trend Vision One console using a corporate identity provider (IdP) solution.")
