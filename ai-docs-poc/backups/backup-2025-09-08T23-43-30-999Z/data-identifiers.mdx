:::: {role="main"}
Digital assets are files and data that an organization must protect against unauthorized transmission. Define digital assets using the following data identifiers:

- *Expressions*: Data with a certain structure.

- *Keywords*: Special words or phrases.

:::note

You cannot delete a data identifier used by a compliance template. Delete the template before deleting the data identifier.
:::
