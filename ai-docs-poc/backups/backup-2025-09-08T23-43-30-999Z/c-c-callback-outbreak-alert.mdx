::: {role="main"}
Configure the following event notification to notify administrators when communications between multiple endpoints and known C&C callback addresses have been detected.

### Procedure {#procedure}

1.  Go to **Detections → Notifications → Event Notifications**.

    The **Event Notifications** screen appears.

2.  Click **Advanced Threat Activity**.

    A list of events appears.

3.  In the *Event* column, click **C&C callback outbreak alert**.

    The **C&C Callback Outbreak Alert** screen appears.

4.  Specify the following notification settings.

    <table>
    <colgroup>
    <col style="width: 36%" />
    <col style="width: 64%" />
    </colgroup>
    <thead>
    <tr>
    <th><p>Settings</p></th>
    <th><p>Description</p></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><p>C&amp;C list source</p></td>
    <td><p>Select one or more C&amp;C list sources.</p></td>
    </tr>
    <tr>
    <td><p>Callback attempts</p></td>
    <td><p>Specify the number of callback attempts.</p></td>
    </tr>
    <tr>
    <td><p>Compromised hosts</p></td>
    <td><p>Specify the number of compromised hosts.</p></td>
    </tr>
    <tr>
    <td><p>Period</p></td>
    <td><p>Specify the period of time.</p></td>
    </tr>
    </tbody>
    </table>

5.  Select recipients for the notification.

    1.  From the **Available Users and Groups** list, select contact groups or user accounts.

    2.  Click **\>**.

        The selected contact groups or user accounts appear in the **Selected Users and Groups** list.

6.  Enable one or more of the following notification methods.

    <table>
    <colgroup>
    <col style="width: 21%" />
    <col style="width: 79%" />
    </colgroup>
    <thead>
    <tr>
    <th><p>Method</p></th>
    <th><p>Description</p></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><p>Email message</p></td>
    <td><p>To customize the email notification template, use supported token variables or modify the text in the <strong>Subject</strong> and <strong>Message</strong> fields.</p>
    <p>For more information, see <a href="trend-vision-one-standard-token-variables">Standard Token Variables</a> and <a href="trend-vision-one-c-c-callback-token-variables">C&amp;C Callback Token Variables</a>.</p></td>
    </tr>
    </tbody>
    </table>

7.  To test if recipients can receive the event notification, click **Test**.

8.  Click **Save**.
:::
