::: {role="main"}
Allows Trend Vision One to collect and analyze more detections and data.

### Procedure {#procedure}

1.  Go to **Administration â†’ Support Settings**.

2.  Enable **Hypersensitive mode**.

    :::note

    To conserve network resources, this feature is automatically disabled after 7 days. Once disabled in Support Settings, hypersensitive mode is disabled in the supported applications.
    :::

    :::warning[Important]

    - Hypersensitive mode might generate false positive detections and increase bandwidth usage.

    - Endpoint Sensors only support hypersensitive mode on Windows agents.
    :::

3.  Enable hypersensitive mode on the desired targets.
