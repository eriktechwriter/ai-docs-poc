::: {role="main"}
View information about global threat actors and emerging threats on the information screen.

The information screen shows detailed information about threat actors or emerging threats selected from the [Threat Insights](threat-insights.mdx) screen.

The following table outlines the information screen:

<table>
<colgroup>
<col style="width: 23%" />
<col style="width: 77%" />
</colgroup>
<thead>
<tr>
<th><p>Section</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Summary</p></td>
<td><p>The section at the top of the screen contains a summary of the selected emerging threat or threat actor.</p>


:::note

<p>Security threats and threat actor groups are often given different names by each security vendor. <strong>AKA</strong> lists known aliases for emerging threats and threat actors.</p>


:::


<p>You can view more detailed information on the following tabs:</p>
<ul>
<li><p><strong>Overview</strong>: This tab displays a detailed history of the selected emerging threat or threat actor.</p></li>
<li><p><strong>Risk Management Guidance</strong>: This tab provides suggestions of how your organization can manage the risks presented by the selected emerging threat or threat actor.</p></li>
<li><p><strong>Threat Hunting Queries</strong>: This tab displays related <a href="trend-vision-one-executing-threat-hunting-queries">threat hunting queries</a> that you can run in <strong>XDR Data Explorer</strong>.</p></li>
</ul></td>
</tr>
<tr>
<td><p>Associated Emerging Threats</p></td>
<td><p>This section displays associated emerging threats, the related impact scope, and the time of the most recent data update.</p></td>
</tr>
<tr>
<td><p>Infection Chain</p></td>
<td><p>This section provides an illustration of the multiple attack tools used to bypass detection tools and deliver a payload.</p></td>
</tr>
<tr>
<td><p>Highlighted MITRE Tactics and Techniques</p></td>
<td><p>This section lists the MITRE tactics and techniques associated with the selected threat or threat actor. Click the tactic or technique name to view more information on the MITRE website.</p></td>
</tr>
<tr>
<td><p>Intelligence Data</p></td>
<td><p>This section collects available threat intelligence from Trend Micro and third-party sources about the selected threat or threat actor.</p>
<p>You can view more detailed information on the following tabs:</p>
<ul>
<li><p><strong>Intelligence Reports</strong>: This tab displays the <a href="trend-vision-one-intelligence-reports">intelligence reports</a> associated with the selected threat or threat actor.</p></li>
<li><p><strong>Tactics, Techniques, and Procedures</strong>: This tab lists the MITRE tactics and techniques associated with the selected threat or threat actor. Click the tactic or technique name to view more information on the MITRE website.</p></li>
<li><p><strong>Tools</strong>: This tab lists software applications exploited by the selected threat or threat actor. Click the tool name to view more information.</p></li>
<li><p><strong>Malware</strong>: This tab shows the malicious software used by the selected threat or threat actor. Click the malware name to view more information.</p></li>
<li><p><strong>CVEs</strong>: This tab displays CVEs associated with the selected threat or threat actor. Click the CVE ID to view more information.</p></li>
<li><p><strong>Indicators</strong>: This tab lists IOCs such as URLs and file hashes associated with the selected threat or threat actor. The selected threat or threat actor may also be associated with other IOCs.</p></li>
<li><p><strong>Associated Threat Actors</strong>: This tab displays the associated threat actors of the selected emerging threat.</p></li>
</ul></td>
</tr>
<tr>
<td><p>Impact Scope</p></td>
<td><p>This section highlights evidence of threats found in your environment.</p>
<p>Click the tabs to display any associated <strong>Workbench Alerts</strong>, as well as <strong>Servers</strong>, <strong>Endpoints</strong>, and <strong>Email Addresses</strong> containing matched IOCs of the selected threat or threat actor.</p></td>
</tr>
</tbody>
</table>
:::
