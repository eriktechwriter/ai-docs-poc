Learn about the views and widgets available on the **Network Overview** dashboard, which provides an at-a-glance perspective into your organization's network security posture.

The following table outlines the default widgets displayed on the **Network Overview** dashboard in the **Network Security** app (**Network Security → Network Overview**).

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Widget</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Detection Highlights (supports all Network Security solutions)</p></td>
<td><p>Displays the number of detections for key network threat types. The detection values displayed are based on the Network Security solution you specify from the <strong>Data Source</strong> pulldown menu (either Deep Discovery Inspector, Virtual Network Sensor, TippingPoint, TippingPoint Network Sensor, or all four) and the time period you specify (the pulldown menu default is <strong>Last 7 days</strong>). The number of Workbench alerts related to these detections is also displayed.</p>
<p>Detection numbers are provided for the following threat types:</p>
<ul>
<li>Command and Control (C&amp;C) communication</li>
<li>Reconnaissance</li>
<li>Ransomware</li>
<li>Exploit attempts</li>
<li>Lateral movement</li>
</ul></td>
</tr>
<tr>
<td><p>Top Affected Hosts/IP Addresses (supports all Network Security solutions)</p></td>
<td><p>Lists the IP addresss and hostnames in your network with the most filter detections and highest severity ratings.</p></td>
</tr>
<tr>
<td><p>MITRE ATT&amp;CK® Mapping for Network (supports all Network Security solutions)</p></td>
<td><p>Makes use of the ATT&amp;CK® tactics and techniques detected across your network.</p></td>
</tr>
<tr>
<td>Scanned Traffic Summary (Deep Discovery Inspector / Virtual Network Sensor tab, TippingPoint tab)</td>
<td><p>Provides an overview of scanned traffic, protocols, and discovered devices in your network environment. The widget supports both Network Inspector and Deep Discovery Inspector version 6.6 or later.</p></td>
</tr>
<tr>
<td>Monitored Network Throughput (Deep Discovery Inspector / Virtual Network Sensor tab)</td>
<td>Provides an overview of the network traffic monitored by Virtual Network Sensor. Enables you to view the 30-day average traffic volume, assess total bandwidth capacity, and track network traffic levels down to the minute both for individual appliances and across the network environment.</td>
</tr>
<tr>
<td>Inspection Events Summary (TippingPoint tab)</td>
<td>Displays the number of hits (Blocked and Permitted) detected in your environment and how you have configured filters to mitigate the risk to your network.</td>
</tr>
<tr>
<td>Virtual Patching Activity (TippingPoint tab)</td>
<td>Displays the CVEs detected in your environment and how you have enabled virtual patching filters to mitigate the risk to your network.</td>
</tr>
</tbody>
</table>
