Sync case data between Trend Vision One Case Management and Jira Cloud.

### Before you begin {#before-you-begin}

Ensure you [configured the Jira Cloud integration](jira-cloud-integration.mdx).

### Procedure {#procedure}

1.  In Trend Vision One, go to **Workflow and Automation â†’ Third-Party Integration**.

2.  Under **Vendor**, select **Atlassian** then click **Trend Vision One Connector for Jira Cloud**.

3.  Click **Create Profile**.

4.  Select the **Case type**.

5.  **Name** the case and provide a **Profile description**.

6.  Provide **Authentication** information:

    - **Host**

    - **User name (Issue reporter)**

    - **API token**

7.  Click **Validate Connection** to verify the connection to Jira.

8.  Configure **Jira settings**:

    1.  Select the **Project**.

    2.  Select one or more **Issue types**.

    3.  Select the **Assignee**, if known.

    4.  Enter the **Reference user**, if appropriate.

9.  Select required **Custom fields**.

    For descriptions of fields in Jira, see the [Jira Cloud documentation.](https://support.atlassian.com/)

10. Select **Optional fields**, if appropriate.

11. Click **Save**.
