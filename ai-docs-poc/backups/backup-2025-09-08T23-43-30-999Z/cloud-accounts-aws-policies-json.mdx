The AWS IAM policies required to allow Cloud Accounts to connect to your AWS account provided in JSON code blocks.

The following JSON code blocks are required when manually connecting your AWS account to the Trend Vision One Cloud Accounts app. You can find these code blocks within the stack template file if you download from Cloud Accounts or use and API to call the template. The policies are not unique to your AWS account or Trend Vision One account. For more information about using the code, see [Adding an AWS Account Manually](adding-aws-account-manually.mdx).

Use the links to jump to the policy code you require:

- Policy 1: `VisionOnePolicyPart1`

- Policy 2: `VisionOnePolicyPart2`

- Policy 3: `VisionOnePolicyPart3`
