You can view the list of available components on the Trend Micro Apex Central server on the **Scheduled Update** and **Manual Update** screens.

The following table describes the component information that displays on the **Scheduled Update** and **Manual Update** screens.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Field</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Category</p></td>
<td><p>Displays the name of a component category</p>
<p>Click <img src="./images/updates_expand=GUID-D176EF44-A17B-449A-8323-A3A884297037=1=en-us=Low.webp" /> to display the list of components in a category.</p></td>
</tr>
<tr>
<td><p>Type</p></td>
<td><p>Displays the component type</p></td>
</tr>
<tr>
<td><p>Current Version</p></td>
<td><p>Displays the last version of the component successfully downloaded by Trend Micro Apex Central</p></td>
</tr>
<tr>
<td><p>Last Download</p></td>
<td><p>Displays the time Trend Micro Apex Central downloaded the <em>Current Version</em> of the component</p></td>
</tr>
<tr>
<td><p>Associated Products</p></td>
<td><p>Displays the name of the managed product or the number of managed products using the component</p>
<p>If more than one managed products use the component, move the mouse cursor over the text to display the list of associated managed products.</p></td>
</tr>
</tbody>
</table>
:::
