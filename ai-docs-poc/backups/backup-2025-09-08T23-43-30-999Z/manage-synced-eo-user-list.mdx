:::: {role="main"}
Continue to synchronize new users or remove synchronized users after you have granted Cloud Email and Collaboration Protection access to Exchange Online.

:::note

This feature is available only if you choose to synchronize selected users during the access granting.
:::

The following table outlines the options available in the **Synchronized User List for Exchange Online** screen.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Option</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Add users to synchronize to the list</p></td>
<td><ol>
<li><p>Click <strong>Add</strong>.</p>


:::note

<p>A maximum of 100 users is supported in the list. The <strong>Add</strong> button is disabled when the limit is reached.</p>


:::

</li>
<li><p>In the <strong>Add User</strong> dialog that appears, specify the users you want to synchronize.</p></li>
<li><p>In the <strong>Available Targets</strong> area, specify individual users or select users from groups.</p>
<ul>
<li><p><strong>By user</strong>: specify the exact user principal name of a user and press <strong>Enter</strong> to verify and display the user name.</p></li>
<li><p><strong>By group</strong>: specify at least the first three characters of the group name and press <strong>Enter</strong> to search for and display the available groups.</p></li>
</ul></li>
<li><p>Select the desired users and click the arrow button to add them to the <strong>Selected Targets</strong> area.</p>
<p>You can synchronize a maximum of 100 users.</p></li>
<li><p>Click <strong>Save</strong>.</p>
<p>The selected users are synchronized to Cloud Email and Collaboration Protection. You can add them as targets in policies to protect their email service.</p></li>
</ol></td>
</tr>
<tr>
<td><p>Delete synchronized users from the list</p></td>
<td><ol>
<li><p>Select one or several users, and click <strong>Delete</strong>.</p></li>
<li><p>Confirm that you understand the impact of deleting synchronized users, and click <strong>Delete</strong>.</p>
<p>Deleting users permanently removes them from Cloud Email and Collaboration Protection, and their email service will no longer be protected. You can still add the users back when needed.</p></li>
</ol></td>
</tr>
<tr>
<td><p>Import users to synchronize to the list</p></td>
<td><ol>
<li><p>Prepare a file for import that meets the following requirements:</p>
<ul>
<li><p>The file must be in .txt format.</p></li>
<li><p>Each line in the file should contain only one User Principal Name (UPN).</p></li>
<li><p>The file can contain a maximum of 100 UPNs.</p></li>
</ul></li>
<li><p>Click <strong>Import</strong>.</p></li>
<li><p>In the <strong>Import users</strong> dialog, click <strong>Select file</strong> and choose a file from your local machine to upload.</p></li>
<li><p>Click <strong>Import</strong>.</p></li>
<li><p>Allow up to 5 minutes for synchronization, and then refresh the screen to view the latest user list.</p>
<p>Duplicate users will be overwritten.</p></li>
</ol></td>
</tr>
</tbody>
</table>
