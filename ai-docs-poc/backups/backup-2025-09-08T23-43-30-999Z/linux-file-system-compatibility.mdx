:::: {role="main"}
Real-time Anti-Malware scans require compatible file system hooks. On Linux, various file systems can be used. Compatible file systems are shown in the following table.

:::note

To protect network file systems, you must select *Enable network directory scan* in the malware scan configuration. For information, see [Scan a network directory (real-time scan only)](configure-malware-scans.mdx).
:::
