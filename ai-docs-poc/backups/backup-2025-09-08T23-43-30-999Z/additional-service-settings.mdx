The Trend Vision One Endpoint Security agent program requires that you enable additional services in order to allow certain features to function properly. The following table describes the available services and the features that require each service.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 25%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr>
<th><p>Service</p></th>
<th><p>Description</p></th>
<th><p>Features</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Unauthorized Change Prevention Service (<code>TMBMSRV.exe</code>)</p></td>
<td><p>Regulates application behavior and verifies program trustworthiness</p></td>
<td><ul>
<li><p>Predictive Machine Learning</p></li>
<li><p>Behavior Monitoring</p></li>
<li><p>Device Control</p></li>
<li><p>Certified Safe Software Service</p></li>
<li><p>Agent Self-protection</p></li>
</ul></td>
</tr>
<tr>
<td><p>Firewall Service (<code>TmPfw.exe</code>)</p></td>
<td><p>Regulates network connection access permissions</p></td>
<td><ul>
<li><p>Apex One Firewall</p></li>
</ul></td>
</tr>
<tr>
<td><p>Suspicious Connection Service</p></td>
<td><p>Provides advanced protection against C&amp;C callbacks</p></td>
<td><ul>
<li><p>User-defined IP Approved and Blocked Lists</p></li>
<li><p>Global C&amp;C IP List (Network Content Inspection Engine)</p></li>
<li><p>Malware network fingerprinting (Relevance Rule Pattern)</p></li>
</ul></td>
</tr>
<tr>
<td><p>Data Protection Service (<code>dsagent.exe</code>)</p></td>
<td><p>Provides advanced monitoring of sensitive data and restricts device access on endpoints</p></td>
<td><ul>
<li><p>Data Loss Prevention</p></li>
<li><p>Device Control (Block access)</p></li>
<li><p>Data Discovery (managed using the Trend Micro Apex Central console)</p></li>
</ul></td>
</tr>
<tr>
<td><p>Advanced Protection Service (<code>TMCCSF.exe)</code>)</p></td>
<td><p>Facilitates advanced scanning and protection features</p></td>
<td><ul>
<li><p>Predictive Machine Learning</p></li>
<li><p>Browser Exploit Prevention</p></li>
<li><p>Behavior Monitoring</p></li>
</ul></td>
</tr>

</table>
