:::: {role="main"}
Manage connected account settings including enabling and disabling cloud security features and permissions.

Clicking on an AWS account name opens the **Cloud Account Settings** screen with two tabs to manage your account.

:::note

Your deployment type changes which tabs appear:

- CloudFormation deployments display **Account Information** and **Stack Update**.

- Terraform deployments display **Account Information** and **Resource Update**.
:::
