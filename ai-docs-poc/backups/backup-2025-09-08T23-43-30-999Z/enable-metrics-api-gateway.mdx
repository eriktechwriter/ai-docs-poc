You can enable detailed metrics for an API Gateway in AWS using the AWS Management Console.

### Procedure {#procedure}

1.  Log in to the AWS Management Console and open the [Amazon API Gateway console](https://console.aws.amazon.com/apigateway).

2.  In the left navigation pane, select an API.

3.  In the **Stages** section, select the stage for which you want to enable detailed metrics.

4.  In the Stage Editor pane, select the **Logs and Tracing** tab, and then click **Edit**.

5.  Select **Detailed metrics**.

6.  Click **Save**.
:::
