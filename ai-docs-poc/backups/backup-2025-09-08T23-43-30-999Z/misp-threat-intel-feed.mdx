The Malware Information Sharing Platform (MISP) can receive events and indicators from the Trend Threat Intelligence feed.

Prerequisites:

- [MISP version 2.4](https://github.com/MISP/MISP/tree/2.4) or later

- [MISP privileges](https://www.circl.lu/doc/misp/administration/)

  - Perm site admin

  - Perm auth

  - Galaxy editor

  - Write access at the operating system level

- [Trend Vision One application programming interface (API) token](api-for-threat-intelligence-feed.mdx)

- [Credits allocated to the Trend Threat Intelligence Feed](credit-req-for-apps-services.mdx)

## Add the feed in MISP {#add-the-feed-in-misp}

Access the Trend Threat Intelligence Feed through the built-in feeds functionality in MISP.

### Procedure {#procedure}

1.  In the MISP web interface, select **Sync Actions â†’ Feeds**.

2.  Click **Add Feed** to display the configuration page.

3.  Select **Enabled**.

4.  Enter the **Name** and **Provider** for the feed.
