### Procedure {#procedure}

1.  Select the following options:

    - **Enable virus/malware scan**

    - **Enable spyware/grayware scan**

      :::note

      You must enable virus/malware scanning before you can enable spyware/grayware scanning.
      :::

2.  Configure the **Target** settings.

    For more information, see [Scheduled Scan: Target Tab](scheduled-scan-targets-tab-apex.mdx).

3.  Configure the **Action** settings.

    For more information, see [Scheduled Scan: Action Tab](scheduled-scan-actions-tab-apex.mdx).

4.  Configure the **Scan Exclusion** settings.

    For more information, see [Scheduled Scan: Scan Exclusion Tab](scheduled-scan-scan-exclusion-tab.mdx).
