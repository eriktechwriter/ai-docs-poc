::: {role="main"}
You can enable scanning in any mount points whose status is as Unmounted. When you enable scanning for a mount point, it scans all of the files in the selected folder. It continues to monitor the folder and scans any files that are added to, or modified in, the folder.

### Procedure {#procedure}

1.  In Trend Vision One go to **Cloud Security â†’ File Security**.

2.  Click **Inventory**.

3.  Click the **Virtual Appliance** node.

4.  Under Actions, click the **Modify Mount Point** (pen) icon for the Service Gateway whose mount you want to disable.

    The **Manage Mount Points for Scanner** drawer opens.

5.  Click the edit (pencil) icon for the **Server IP**.

6.  Enable **Scanning**.

7.  Click **Save**.
:::
