Move agents from Trend Vision One Endpoint Security.

The following steps detail how to move agents from Trend Vision One Endpoint Security to Trend Cloud One - Endpoint & Workload Security.

Before starting, you must deactivate the target agent you want to move.

- For information on how to deactivate agents, see the "Deactivate the agent" section in [Activate the agent](activate-agent.mdx).

- You may also need to disable agent self-protection before you can deactivate the agent. For more information, see [Enable or disable agent self-protection](use-agent-self-protection.mdx).

### Procedure {#procedure}

1.  In the Trend Cloud One - Endpoint & Workload Security console, go to **Administration → Updates → Software → Local**.

2.  Click **Generate Deployment Scripts**.

3.  Select the **Platform** and configure the other settings.

4.  Copy the activation command from the deployment script.

    The activation command is the line which includes the manager URL, port, `tenantID`, and `token`.

5.  Access the device where the target agent is installed.

6.  Run Command Prompt as an Administrator.

7.  Run the activation command you copied from the deployment script.

8.  After reactivating the agent, ensure the correct policies and configurations are applied.
