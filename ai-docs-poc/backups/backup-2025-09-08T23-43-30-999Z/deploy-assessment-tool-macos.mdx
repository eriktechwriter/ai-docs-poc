Deploy the assessment tool to scan macOS endpoints for suspicious file activity and file-based threat indicators from global intelligence sources.

The tool supports macOS 10.11 or later.

Endpoints must be connected to the internet so the tool can upload data to Trend Vision One. Trend Micro recommends adding the following URLs and ports to the safe list.

The assessment report identifies unknown threats and helps determine if your organization is affected by the latest threat campaigns.

### Procedure {#procedure}

1.  In the **At-Risk Endpoint Assessment** area, click **Start Assessment**.

2.  Click **Download Assessment Tool**. Click **Download Assessment Tool** under **macOS** on the dialog that displays.

    :::note

    The assessment tool continuously collects user activity, cloud app access, and vulnerability data to provide advanced risk visibility.

    To uninstall the tool from your endpoints, contact your support provider.
    :::

3.  Locate and double-click the downloaded file (`endpoint_basecamp_mac.zip`).

    A window opens.

4.  Double-click `endpoint_basecamp.pkg`.

    The **Install Endpoint Basecamp** window opens.

5.  Complete the steps in the installation wizard.

6.  Click **Close**.

    The tool starts running in the background and automatically uploads data to Trend Vision One.
:::
