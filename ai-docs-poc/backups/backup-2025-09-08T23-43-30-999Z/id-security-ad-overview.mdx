View identity provider status, key metrics, and critical alerts related to Active Directory.

The **Overview** tab displays a summary of your connected identity provider status along with at-a-glance metrics on your accounts and objects.

The following table describes the information categories available in the **IDENTITY PROVIDER SUMMARY (ALL DOMAINS)** section.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Category</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Data synchronization</p></td>
<td><p>Displays the status of the configuration required to sync identity data</p></td>
</tr>
<tr>
<td><p>User access control</p></td>
<td><p>Displays the status of the configuration required to perform user access control actions</p></td>
</tr>
<tr>
<td><p>Security event forwarding</p></td>
<td><p>Displays the status of the configuration required to forward security events</p></td>
</tr>
</tbody>
</table>
