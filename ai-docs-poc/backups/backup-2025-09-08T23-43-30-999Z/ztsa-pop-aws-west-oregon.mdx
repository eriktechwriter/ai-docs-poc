August 12, 2024—Zero Trust Secure Access Internet Access now offers support for the AWS US West (Oregon) region. Users in the region may configure their service FQDNs to reflect the new location. For more information on available PoP sites for the Internet Access Cloud Gateway, see [Port and FQDN/IP address requirements](https://docs.trendmicro.com/en-us/documentation/article/trend-vision-one-port-fqdn-ip-address-requirements).
