:::: {role="main"}
View the supported operating systems and their system requirements needed to deploy the Trend Vision One Endpoint Security agent with only the Endpoint Sensor.

:::warning[Important]

The Endpoint Sensor currently does not support Windows 11 ARM64. Deploy the Trend Vision One Endpoint Security agent with Standard Endpoint Protection to provide monitoring and support for Windows 11 ARM64. Endpoint Sensor support is coming soon.
:::
