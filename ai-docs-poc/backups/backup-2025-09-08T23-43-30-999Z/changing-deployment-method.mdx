Adjust the **Mobile Security** deployment method to suit your organization's latest IT infrastructure.

### Procedure {#procedure}

1.  Go to **Mobile Security â†’ Mobile Inventory**.

2.  Click the **Settings** icon (![](/images/Global_Settings=GUID-1E10BFBD-3AFF-46DD-B853-0438EC2FD3F9.webp)) at the upper-right corner.

3.  Disconnect from Trend Vision One by clicking **Reset** at the bottom of the screen.

    :::warning[Important]

    Resetting the current connection erases all existing user and device data and this action is irreversible.
    :::

    The **Mobile Security** landing page will appear after a few seconds.

4.  Select a deployment method, and establish initial configuration to reconnect to Trend Vision One and enroll devices.

    [Learn about the initial configuration steps](getting-started-mobile-security.mdx)
:::
