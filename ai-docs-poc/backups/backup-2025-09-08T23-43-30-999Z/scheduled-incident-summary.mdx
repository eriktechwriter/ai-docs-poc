::: {role="main"}
Configure the following event notification to send administrators a summary of the DLP incidents that occurred on your network.

### Procedure {#procedure}

1.  Go to **Detections → Notifications → Event Notifications**.

    The **Event Notifications** screen appears.

2.  Click **Data Loss Prevention**.

    A list of events appears.

3.  In the *Event* column, click **Scheduled incident summary**.

    The **Scheduled Incident Summary** screen appears.

4.  Specify the following notification settings.

    <table>
    <colgroup>
    <col style="width: 36%" />
    <col style="width: 64%" />
    </colgroup>
    <thead>
    <tr>
    <th><p>Criteria</p></th>
    <th><p>Description</p></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><p>Frequency</p></td>
    <td><p>Select to receive notifications on a daily or weekly basis.</p></td>
    </tr>
    <tr>
    <td><p>Attach incident details</p></td>
    <td><p>Select to attach incident logs to the notification.</p>
    <ul>
    <li><p>Select the content that DLP Compliance Officers receive:</p>
    <ul>
    <li><p><strong>Incidents from all managed users</strong></p></li>
    <li><p><strong>Incidents from direct reports only</strong></p></li>
    </ul>
    

    :::note
    
    <p>DLP Incident Reviewers can only receive incidents from direct reports</p>
    

    :::

    </li>
    <li><p>Select the format of the log details</p></li>
    </ul></td>
    </tr>
    </tbody>
    </table>

5.  Enable one or more of the following notification methods.

    <table>
    <colgroup>
    <col style="width: 21%" />
    <col style="width: 79%" />
    </colgroup>
    <thead>
    <tr>
    <th><p>Method</p></th>
    <th><p>Description</p></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><p>Email message</p></td>
    <td><p>To customize the email notification template, use supported token variables or modify the text in the <strong>Subject</strong> and <strong>Message</strong> fields.</p>
    <p>For more information, see <a href="trend-vision-one-standard-token-variables">Standard Token Variables</a> and <a href="trend-vision-one-dlp-token-variables">Data Loss Prevention Token Variables</a>.</p></td>
    </tr>
    </tbody>
    </table>

6.  To test if recipients can receive the event notification, click **Test**.

7.  Click **Save**.
:::
