Generates and views one-time and scheduled reports.

Reports are generated using the **Threat Detection** template that summarizes threat detection in your services by threat type.

The following table outlines the tabs on the **Reports** screen (Email and Collaboration Security Operations → Cloud Email and Collaboration Protection → Reports).

<col style="width: 75%" />
</colgroup>

<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p><strong>Generated Reports</strong></p></td>
<td><p>Views all previously generated reports and status information, and downloads and deletes reports.</p></td>
</tr>
<tr>
<td><p><strong>Scheduled Reports</strong></p></td>
<td><p>Creates, views, edits, and deletes the schedules for scheduled reports.</p>
<p>For more information about generating scheduled reports, see <a href="trend-vision-one-configuring-reports">Configuring reports</a>.</p></td>
</tr>
<tr>
