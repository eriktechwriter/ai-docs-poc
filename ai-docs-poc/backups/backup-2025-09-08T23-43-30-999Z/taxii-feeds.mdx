:::: {role="main"}
Trend Vision One allows you to subscribe to and monitor TAXII feeds for threat information that can be used to complement custom intelligence.

:::note

Only users with the Master Administrator role can access the **TAXII Feeds** screen.
:::

The following table outlines the actions available on the **TAXII Feeds** screen.

<tr>
