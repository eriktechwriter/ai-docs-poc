:::: {role="main"}
Use Microsoft Intune to deploy the agent package to your managed endpoints.

:::warning[Important]

Agent deployment using Microsoft Intune has only been tested with Windows agents.
:::

Before you get started, make sure to download and unzip the [Microsoft Win32 Content Prep Tool](https://github.com/Microsoft/Microsoft-Win32-Content-Prep-Tool).

Refer to the following topics depending on your deployment type.

- [Standard Endpoint Protection Agent Deployment using Microsoft Intune](sep-agent-deploy-intune.mdx)

- [Server & Workload Protection Agent Deployment using Microsoft Intune](swp-agent-deploy-microsoft-intune.mdx)

- [Endpoint Sensor Agent Deployment using Microsoft Intune](es-agent-deploy-intune.mdx)

- To uninstall agents, see [Uninstall Windows Agents with Microsoft Intune](uninstall-agents-intune.mdx).
