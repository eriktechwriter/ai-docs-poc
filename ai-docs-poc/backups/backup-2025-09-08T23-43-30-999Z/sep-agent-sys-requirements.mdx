:::: {role="main"}
View the supported operating systems and their system requirements needed to install the Trend Vision One Endpoint Security agent with Standard Endpoint Protection features.

:::warning[Important]

The Endpoint Sensor currently does not support Windows 11 ARM64. You can still deploy the Trend Vision One Endpoint Security agent with Standard Endpoint Protection package to monitor your Windows 11 ARM64 endpoints. However, the Endpoint sensor detection and response feature cannot be enabled at this time.
:::
