Some versions of Deep Security Agent for Linux support [Secure Boot](https://uefi.org/sites/default/files/resources/UEFI_Secure_Boot_in_Modern_Computer_Security_Solutions_2013.pdf), as shown in the following table. See also [Configure Linux Secure Boot for agents](config-linux-secure-boot-agents.mdx).

Each operating system is associated with corresponding Secure Boot public keys, such as `DS2022.der`, `DS20.der`, and so on. These keys have different expiration dates. For more information, see [The public key has expired](config-linux-secure-boot-agents.mdx).

Deep Security Agent does not currently support Secure Boot on Azure virtual machines.

Indicators:

✔ - Supported. If an update added support, then the minimum required version appears in the footnote.

• - Support for these releases is ending soon. Please upgrade as soon as possible.
