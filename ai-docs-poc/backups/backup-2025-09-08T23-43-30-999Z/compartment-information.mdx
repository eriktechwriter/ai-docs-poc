::: {role="main"}
Configure basic settings for your connected Oracle Cloud compartments.

The Compartment Information tab in the Compartment settings screen allows you to view and configure basic settings. After changing a field, click **Save Changes**. Updating information on the Compartment Information tab does not require performing a resource update.

<table>
<thead>
<tr>
<th><p>Field</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>OCID</p></td>
<td><p>The Oracle Cloud Identifier of the connected Oracle Cloud compartment.</p>
<p>This value is provided when connecting the compartment to Trend Vision One and cannot be changed.</p></td>
</tr>
<tr>
<td><p>Identity Domain OCID</p></td>
<td><p>The Identity Domain Oracle Cloud Identifier of the compartment's identity domain.</p>
<p>This value is provided when connecting the compartment to Trend Vision One and cannot be changed.</p></td>
</tr>
<tr>
<td><p>Name</p></td>
<td><p>The name of the compartment as displayed in Cloud Accounts.</p></td>
</tr>
<tr>
<td><p>Description</p></td>
<td><p>The description of the compartment as displayed in Cloud Accounts.</p></td>
</tr>
</tbody>
</table>
:::
