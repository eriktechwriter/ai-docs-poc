Create a data policy, which specifies a network location containing local devices and files that you want monitored for threats and data leakage.

### Procedure {#procedure}

1.  In the Trend Vision One console, go to **Data Security** \> **Data Policy**.

2.  Click **Create Policy**.

3.  Provide a name for the policy.

4.  In the **Network location** field, enter the Uniform Resource Identifier (URI) of the network location.

5.  Optionally, add a description of the data policy.

6.  To apply the policy only to files in the network location with specific data attributes:

    1.  Enable **Advanced Settings**.

    2.  Clear the checkbox beside each file type you want to exclude from the policy.

7.  Click **Save**.

The policy is added to the list in the **Data Policy** page.
:::
