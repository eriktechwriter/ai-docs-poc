Data Loss Prevention can monitor the following system and application channels:

- Cloud storage services

- Data recorders (CD/DVD)

- Peer-to-peer applications

- PGP Encryption

- Printer

- Removable storage

- Synchronization software (ActiveSync)

- Windows clipboard
