After subscribing to the Trend Vision One Endpoint Security agent on the Amazon Web Services (AWS) marketplace, you can use the agent to create an Amazon Machine Image (AMI) to start and connect an Amazon Elastic Compute Cloud (EC2) to Trend Vision One.

### Procedure {#procedure}

1.  [Retrieve the Trend Vision One parameters](retrieve-v1-parameters.mdx).

2.  [Set parameter store values](set-parameter-store-values.mdx).

3.  [Set up the image builder and create an Amazon Elastic Compute Cloud (EC2) instance](image-builder-ec2-instance.mdx).
