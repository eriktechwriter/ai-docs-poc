### Procedure {#procedure}

1.  Go to **Dashboard**.

2.  Click the **Summary** tab.

3.  Scroll down and find the **Product Connection Status** widget.

    You can check the status of any Cloud Email Gateway Protection server registered with Trend Micro Apex Central.
