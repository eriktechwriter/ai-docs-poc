### Procedure {#procedure}

1.  Go to **Directories â†’ Users/Endpoints**.

2.  Expand the **Labels** node under **Endpoints** in the tree.

3.  Click **Add new label**.

    The **Label Settings** screen appears.

4.  Type a descriptive name for the label.

5.  To create an auto-label rule:

    1.  Click **Create auto-label rules**.

    2.  Specify the criteria for the rule.

6.  Click <span class="uicontrol keyboard">Save</span>.

    The custom label appears under the **Labels** node.

    After creating a custom label:

    - Click the ![](/images/edit_tag_filter=20220412173608=.webp) icon next to any custom label to rename the label.

    - Click the ![](/images/trash=20220412173430=.webp) icon next to any custom label to delete the label.

      :::note

      You cannot delete labels that have been specified as policy targets.
      :::

    - Click the ![](/images/update_filter=20220412173255=.webp) icon next to any custom label to edit the auto-label rule criteria.
:::
