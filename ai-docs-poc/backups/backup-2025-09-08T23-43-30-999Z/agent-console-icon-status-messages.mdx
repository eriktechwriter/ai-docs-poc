A list of agent console status messages and how to troubleshoot potential issues.

The Trend Vision One Endpoint Security agent console provides endpoint users with a quick way to view the status of the agent. The system tray icon displays different symbols depending on the status, and the user can hover the mouse over the icon to see a status message.

<td><p>Trend Vision One Endpoint Security is online</p></td>
<td><p>The agent is operating normally and successfully connected to Trend Vision One servers</p></td>

</tr>
<tr>
<td><p>Trend Vision One Endpoint Security is waiting for information</p></td>
<td><p>The agent is operating normally and is connecting or attempting to connect to Trend Vision One servers</p></td>
<td><p>This status is normal behavior. If the agent takes a long time to connect, check your network settings. Make sure the endpoint can connect to the internet, or that the agent can connect to a Service Gateway.</p></td>
</tr>
<tr>
<td><p>Trend Vision One Endpoint Security is offline</p></td>
<td><p>The agent is unable to connect to Trend Vision One servers</p>
<p>The agent is unable to retrieve updated security settings or download software and component updates.</p></td>
<td><p>Check your network settings. Make sure the endpoint can connect to the internet, or that the agent can connect to a Service Gateway.</p></td>
</tr>
<tr>
<td><p>Your device is isolated</p></td>
<td><p>The Isolate Endpoint response action has been applied to the endpoint</p></td>
<td><p>Investigate if the endpoint is safe to remove from isolation. If you determine the endpoint is safe to remove from isolation, <a href="trend-vision-one-restore-connection-task">restore the connection</a>.</p></td>
</tr>
<tr>
<td><p>Trend Vision One Endpoint Security console is disabled by Admin</p></td>
<td><p>The agent console is disabled in Endpoint Security Policies</p>
<p>The system tray icon still appears to let you know the agent is running, but endpoint users cannot access the console or receive notifications.</p></td>
<td><p>If you want to allow users to access the agent console and receive notifications, enable <strong>Agent Console</strong> in your <a href="trend-vision-one-configuring-endpoint-policies">Endpoint Security policy configuration</a>.</p></td>
</tr>
