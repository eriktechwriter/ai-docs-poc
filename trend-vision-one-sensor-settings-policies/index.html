Endpoint Security Policies

Create and configure policies to centrally manage endpoint agent settings.

Important

  • The first time you access the Endpoint Security Policies screen, the system imports your sensor settings from Endpoint Inventory to create several general policies along with the Default Endpoint Policy. For more information, see About endpoint security policies.

    You might also see a message about the Network Content Inspection Engine feature. For more information, see Network Content Inspection Engine.

  • Endpoint security policies only support endpoints with the Trend Vision One Endpoint Security agent version October 2024 release or later installed.

Use the Endpoint Security Policies screen (Endpoint Security → Endpoint Security Configuration → Endpoint Security Policies) to create policies which enable you to manage settings for your deployed Trend Vision One Endpoint Security agent, including enabling endpoint detection and response. For detailed information about what endpoint security policies manage, see About endpoint security policies.

The following table details the actions you can take on the Endpoint Security Policies screen:

Action

Details

Create an endpoint security policy

Click Add Policy to create a new endpoint security policy

Edit an existing policy

Click the policy name to edit an existing endpoint security policy

Locate a policy

Use the search field to locate the name of an endpoint security policy or a priority level

Delete a policy

Select one or more policies and click the Delete button

note

Deleting a policy automatically reassigns endpoint groups within that policy to the Default Endpoint Policy.

You cannot delete the Default Endpoint Policy.
