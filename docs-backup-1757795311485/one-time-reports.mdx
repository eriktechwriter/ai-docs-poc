# One-time Reports

The **One-time Reports** screen provides a list of all previously generated one-time reports for your network. You can use this screen to create new one-time reports and view previously generated one-time reports.

The following table outlines the available tasks on the **One-time Reports** screen.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Task</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Add new one-time reports</p></td>
<td><p>Click <strong>Add</strong> to create a new one-time report.</p>
<p>For more information, see <a href="trend-vision-one-creating-one-time-reports">Creating One-time Reports</a>.</p></td>
</tr>
<tr>
<td><p>Delete one-time reports</p></td>
<td><p>Select an existing one-time report and click <strong>Delete</strong>.</p></td>
</tr>
<tr>
<td><p>Forward one-time reports to email recipients</p></td>
<td><p>Select an existing one-time report and click <strong>Forward</strong> to email the report as an attachment to specified recipients.</p></td>
</tr>
<tr>
<td><p>View generated one-time reports</p></td>
<td><p>Click the <strong>View</strong> link in the <em>View</em> column of the report you want to view.</p></td>
</tr>
<tr>
<td><p>View one-time report profiles</p></td>
<td><p>Click the <em>Name</em> of a previously generated one-time report to view the report profile.</p>


:::note

<p>You cannot edit the profiles of previously generated one-time reports.</p>


:::

</td>
</tr>
</tbody>
</table>
