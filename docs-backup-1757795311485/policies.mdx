# Policies

Define Container Protection policies to ensure protection and vulnerability detection for your containers during deployment and run time.

Container Protection allows you to define cluster-wide policy rules and more granular, prioritized namespace rules to ensure that you can configure optimal protection to all of your Kubernetes containers during deployment and run time. Create and assign rulesets to both your Kubernetes and Amazon ECS clusters to ensure run time protection for all your containers.
