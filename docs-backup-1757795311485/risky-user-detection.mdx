# Risky user detection

Learn about the widgets that show the top risky users and user risk trends in your organization.

The **Risky User Detection** widgets present internal user accounts most at risk and user risk trends based on user behavior analysis.

Internal user accounts are the accounts that belong to the internal domains defined in global settings. For details, see [Configuring the internal domain list](configuring-internal-domain-list.mdx).
