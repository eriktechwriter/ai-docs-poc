# Standard Endpoint Protection, Server & Workload Protection, and Endpoint Sensor now support Windows 11 (24H2)

November 29, 2024—The Trend Vision One Endpoint Security agent now supports Windows 11 (24H2) for Standard Endpoint Protection, Server & Workload Protection, and Endpoint Sensor-only deployments. For details on supported Windows platforms, see [Trend Vision One Endpoint Security agent system requirements](endpoint-agent-system-requirements.mdx).
