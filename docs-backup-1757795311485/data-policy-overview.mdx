# Data Policy

Define data policies for Data Security to identify the network locations with local devices and files that contain sensitive data.

A data policy specifies the network location and, optionally, the file types in that location that contain sensitive data. Data Security uses the data policies you define to create an inventory of assets and files with sensitive data, which you can view in the Data Inventory app.

:::note

Before you begin, note the following:

- You must deploy endpoint data sensors in the network locations that have a data policy to enable Data Detection and Response. For more information, see [Configure endpoint security policies](configuring-endpoint-policies.mdx).

- Data policies monitor local devices and files, not cloud assets. To monitor cloud assets for sensitive data, you must enable [Data Security Posture](data-posture.mdx).
:::
