# Summary widget

For a selected [Account](cloud-accounts-cloud-posture-setup.mdx), or [Group](grouped-accounts.mdx), the summary widget of the [Misconfiguration and Compliance](cloud-identity-dashboard-view.mdx) provides:

1.  [Compliance Score](compliance-score.mdx)
2.  [Check](model-check.mdx) summaries broken down successes, failures, and different severities

From the summary widget, you can action or access:

- Improve
- Browse all checks
- Run the Cloud Risk Management Bot
- Update tags
- Reports
- Settings

### Improve {#improve}

Report of all failed Checks for the account.

### Browse all checks {#browse-all-checks}

Access [All Checks Report](all-checks-report.mdx) which details all provides a list of all Check failures and successes for the account.

### Run the Cloud Risk Management Scan {#run-the-cloud-risk-management-scan}

By default, the [Cloud Risk Management Scan](cloudIdentityBot.mdx) runs once every hour. You can run the Cloud Risk Management Scan on demand by clicking on "Run the Cloud Risk Management Scan" again.

### Update tags {#update-tags}

Organize your accounts into [Groups](grouped-accounts.mdx) using [Tags](cloud-account-tags.mdx). This functionality is available only when viewing an individually selected [Account](cloud-accounts-cloud-posture-setup.mdx).

### Reports {#reports}

Create, schedule, save and access [Configured Reports](configured-reports.mdx).

### Settings {#settings}

See:

- [account settings](cloud-acct-cloud-posture.mdx) when viewing a single [Account](cloud-accounts-cloud-posture-setup.mdx)
- [group settings](group-settings.mdx) when viewing a [Group](grouped-accounts.mdx)
:::
