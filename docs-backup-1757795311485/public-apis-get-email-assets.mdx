# Trend Vision One public APIs available to retrieve email assets

December 13, 2024—You can now use the Trend Vision One RESTful APIs to get managed email accounts, email domains, and email servers from Email Asset Inventory.

For more information, see [Trend Vision One Automation Center](https://automation.trendmicro.com/xdr/api-v3/).

**Email and Collaboration Security → Email Asset Inventory**
