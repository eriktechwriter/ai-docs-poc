# Configuring spam criteria

### Procedure {#procedure}

1.  Select "**Spam**".

2.  Choose a baseline spam catch rate.

    - Lowest (most conservative)

    - Low

    - Moderately low (the default setting)

    - Moderately high

    - High

    - Highest (most aggressive)
:::
