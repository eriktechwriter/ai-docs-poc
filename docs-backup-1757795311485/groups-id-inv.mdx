# Group

View information on the groups in your organization, including type and privilege status.

The **Group** tab allows you to view current groups in your organization. Groups consist of users, service principals, devices, and other groups that have a specific access level or set of permissions.

The following table details the columns on the **Group** tab.

<td><p>Indicates whether a group is privileged.</p>

</tr>
<tr>

<td><p>Type of a group.</p>
<ul>

<li><p>Mail-enabled security: functions like security groups but also allows email communication.</p></li>
<li><p>Distribution: intended for sending notifications to a group of users.</p></li>
