# Email communication

Trend Vision One™ – Cloud Risk Management supports notifications sent via Email.

### Set up an Email channel {#set-up-an-email-channel}

1.  Click on *Create another Email channel*
2.  [Set automatic notifications](toggle-automatic-notifications.mdx)
3.  [Configure Triggers](communication-triggers.mdx)
4.  [Configure Users](communication-recipients.mdx)
5.  [Copy communication settings](copy-communication-settings.mdx)

:::note

Active users without an email will not appear on the recipient list.
:::
::::
