# Logs

Analyze your system security and tailor cloud application and service protection by querying and viewing log information.

Cloud Email and Collaboration Protection keeps detailed logs about security detection and system event data in the management console, such as advanced threat protection and data loss prevention scanning, quarantined messages, and activities affecting the management console. For details, see [Log types](log-types.mdx).

The following table outlines the actions available on the **Logs** screen (Email and Collaboration Security Operations → Cloud Email and Collaboration Protection → Logs).
