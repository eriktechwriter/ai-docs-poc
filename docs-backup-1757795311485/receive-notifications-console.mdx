# Receiving notifications from the Trend Vision One console

Configure Trend Vision One to send notifications and alerts to the mobile app.

### Procedure {#procedure}

1.  Make sure that you have configured alerts and notifications in the Trend Vision One console and selected the mobile app as the notification channel.

    For details, see [Configuring notifications](configure-admin-notifications.mdx).

2.  Configure Trend Vision One Mobile:

    1.  In the Trend Vision One Mobile app, go to **Settings () â†’ Notification**.

    2.  Enable the notifications you configured in the web console.

3.  In the settings app of your device, make sure Trend Vision One Mobile has permission to send notifications.
