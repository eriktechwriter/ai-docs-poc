# Policies

Define security policies for Cloud Email and Collaboration Protection to detect and take action on threats and data leakage.

A policy is a set of rules that Cloud Email and Collaboration Protection uses to evaluate emails, files, chat messages, or object records in your protected services. You can configure policies to determine the actions to apply to detected threats or data leakage.
