# Managing recipient filter

The **Recipient Filter** screen displays the list of available domains. You can enable or disable these domains to check valid recipients and export the domain recipient lists to local storage.

**Recipient Filter Tasks**

<table>
<colgroup>
<col style="width: 28%" />
<col style="width: 72%" />

<thead>
<tr>
<th><p>Tasks</p></th>
<th><p>Steps</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Enable All Filters</p></td>
<td><p>On the <strong>Recipient Filter</strong> screen, click <strong>Enable All</strong> to enable all filters in all domains.</p></td>
</tr>
<tr>
<td><p>Disable All Filters</p></td>
<td><p>On the <strong>Recipient Filter</strong> screen, click <strong>Disable All</strong> to disable all filters in all domains.</p></td>
</tr>
<tr>
<td><p>Export All</p></td>
<td><p>On the <strong>Recipient Filter</strong> screen, click <strong>Export All</strong> to export all filters in all domains to the local storage.</p></td>
</tr>
<tr>
<td><p>Export A Filter List</p></td>
<td><p>On the <strong>Recipient Filter</strong> screen, click the <img src="./images/export_icon=GUID-A28E077D-F629-488B-8C7B-28652EF6C808=1=en-us=Low.webp" /> icon under the <strong>Export</strong> column to export the filter list in a domain.</p></td>
