# Configure IMDSv2 from deployed EC2 connectors

### Procedure {#procedure}

1.  Log into your AWS console and find the EC2 instance of your connector.

2.  Go to **Actions**, and select **Instance Settings**.

3.  Select **Modify instance Metadata options**.

4.  Change the **IMDSv2** option to Required for all EC2 connectors.

5.  Click **Save**.
:::
