Besides the preceding default policy rules, Cloud Email Gateway Protection also presets a built-in policy rule "Global Anti-Virus Rule (Enforced on Unverified Domains)", which is forcibly applied to inbound messages sent to unverified domains.

:::note

This policy rule does not appear on the policy screen, and is visible only in mail tracking logs, policy event logs, and quarantine query details.
:::
