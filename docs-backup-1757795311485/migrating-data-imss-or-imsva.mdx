# Migrating data from IMSS or IMSVA

If you are a customer of InterScan Messaging Security Suite (IMSS) or InterScan Messaging Security Virtual Appliance (IMSVA) and want to switch to Cloud Email Gateway Protection, Cloud Email Gateway Protection allows you to migrate your existing data from IMSS 9.1 or IMSVA 9.1.
