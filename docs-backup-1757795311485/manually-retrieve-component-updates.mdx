# Manually retrieve component updates

::::tip

You can configure a scheduled task to automatically check for component updates. For more information, see [Schedule Server & Workload Protection to perform tasks](schedule-perform-tasks.mdx).
:::

Use the following steps to manually retrieve component updates on a single agent:

:::warning[Important]

Deep Security Agent version 20.0.0-179 and later for macOS supports rolling back a component update. Agents for macOS earlier than version 20.0.0-179 do not support rolling back a component update.
:::

### Procedure {#procedure}

1.  Go to **Computers**.

2.  Select the endpoint you want to update.

3.  Right-click and select **Actions â†’ Download Component Update**.
