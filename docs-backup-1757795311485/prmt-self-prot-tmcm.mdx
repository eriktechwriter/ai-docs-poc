# Security Agent Self-protection

Trend Vision One Endpoint Security agent self-protection provides ways for the Trend Vision One Endpoint Security agent to protect the processes and other resources required to function properly. Trend Vision One Endpoint Security agent self-protection helps thwart attempts by programs or actual users to disable anti-malware protection.
