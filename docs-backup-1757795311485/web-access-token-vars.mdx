# Web Access Policy Violation Token Variables

The following table describes token variables for customizing Web Access Policy Violation event notification messages.

:::note

For the list of standard token variables supported by all event notifications, see [Standard Token Variables](standard-token-variables.mdx).
:::

<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />

<th><p>Description</p></th>
</tr>
</thead>

</tr>

</tr>
