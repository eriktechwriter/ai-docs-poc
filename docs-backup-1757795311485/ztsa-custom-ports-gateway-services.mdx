# Configure custom ports for Internet Access On-Premises Gateway services

March 25, 2024—Users may now change the default ports for services such as data proxy, authentication proxy, and ICAP/ICAPS services configured on the Internet Access On-Premises Gateway. Configure custom ports from Service Gateway Management. For more information, see [Service Gateway services](service-gateway-services.mdx).

**Zero Trust Secure Access → Secure Access Configuration → Internet Access and AI Service Access Configuration**
