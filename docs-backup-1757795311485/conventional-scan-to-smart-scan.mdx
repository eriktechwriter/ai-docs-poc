# Switching from Conventional Scan to Smart Scan

If you are switching Trend Vision One Endpoint Security agents from conventional scan to smart scan, ensure that you have set up Smart Protection Services on the Apex One server. For details, see the Apex One documentation.

The following table provides other considerations when switching Trend Vision One Endpoint Security agent to smart scan.

The Apex One (Mac) server can perform other critical tasks while Trend Vision One Endpoint Security agents change their scan methods.</p></td>
</tr>
<tr>
<td style="border-bottom-width: 1px; border-bottom-style: solid; border-right-width: 1px; border-right-style: solid"><p>Timing</p></td>
<td style="border-bottom-width: 1px; border-bottom-style: solid"><p>When switching to smart scan for the first time, Trend Vision One Endpoint Security agents need to download the full version of the Mac Heuristic Pattern and Smart Scan Agent Pattern from the Apex One (Mac) server. The Smart Scan Pattern is only used by smart scan agents.</p>
<p>Consider switching during off-peak hours to ensure the download process finishes within a short amount of time. Also consider switching when no Trend Vision One Endpoint Security agent is scheduled to update from the server.</p></td>
</tr>
<tr>
<td style="border-bottom-width: 1px; border-bottom-style: solid; border-right-width: 1px; border-right-style: solid"><p>Agent tree settings</p></td>
<td style="border-bottom-width: 1px; border-bottom-style: solid"><p>Scan method is a granular setting that can be set on the root, group, or individual agent level. When switching to smart scan, you can:</p>
<ul>
<li><p>Create a new group and assign smart scan as its scan method. Any Trend Vision One Endpoint Security agent you move to this group will use smart scan. When you move the Trend Vision One Endpoint Security agent, enable the setting <strong>Apply settings of new group to selected agent(s)</strong>.</p></li>
<li><p>Select a group and configure it to use smart scan. Conventional scan agents belonging to the group will switch to smart scan.</p></li>
<li><p>Select one or several conventional scan agents from a group and then switch them to smart scan.</p></li>
</ul>


:::note

<p>Any changes to the group's scan method overrides the scan method you have configured for individual Trend Vision One Endpoint Security agents.</p>


:::

</td>

<td style="border-right-width: 1px; border-right-style: solid"><p>IPv6 support</p></td>
<td><p>Smart scan agents send scan queries to smart protection sources.</p>
<p>A pure IPv6 smart scan agent cannot send queries directly to pure IPv4 sources, such as:</p>
<ul>
<li><p>Smart Protection Server 3.0 (integrated or standalone)</p></li>
<li><p>Trend Micro Smart Protection Network</p></li>
</ul>
<p>Similarly, a pure IPv4 smart scan agent cannot send queries to pure IPv6 Smart Protection Servers.</p>
<p>A dual-stack proxy server that can convert IP addresses, such as DeleGate, is required to allow smart scan agents to connect to the sources.</p></td>
