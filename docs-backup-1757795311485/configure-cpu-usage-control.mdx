# Configure CPU usage control

Manage the impact on processing resources by Server & Workload Protection agent features.

:::warning[Important]

CPU usage control only supports Linux agents.
:::

Use the **CPU Usage Control** setting to manage the impact of the endpoint agent on the endpoint processing resources. Use the following steps to configure the setting:

### Procedure {#procedure}

1.  Open the details screen of the Computer you want to configure.

2.  Go to **Settings â†’ General**.

3.  Locate the **CPU Usage Control** setting and select the CPU protection mode you want to use.

    The settings you choose impact the behavior of certain features. Review each setting to determine the best setting for your environment.
