# Incident Details Updated

Configure the following event notification to notify administrators when incident details have been updated.

### Procedure {#procedure}

1.  Go to **Detections → Notifications → Event Notifications**.

    The **Event Notifications** screen appears.

2.  Click **Data Loss Prevention**.

    A list of events appears.

3.  In the *Event* column, click **Incident details updated**.

    The **Incident Details Updated** screen appears.

4.  Specify the criteria for the incident detail updates to be notified about:

    <table>
    <colgroup>
    <col style="width: 36%" />
    <col style="width: 64%" />
    </colgroup>
    <thead>
    <tr>
    <th><p>Criteria</p></th>
    <th><p>Description</p></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><p>Incident detail updates</p></td>
    <td><p>Select the type of incident detail updates.</p>
    <ul>
    <li><p><strong>Closed</strong></p></li>
    <li><p><strong>Any change</strong></p></li>
    </ul></td>
    </tr>
    <tr>
    <td><p>Filter by severity level</p></td>
    <td><p>Select one of more of the following risk levels.</p>
    <ul>
    <li><p><strong>High</strong></p></li>
    <li><p><strong>Medium</strong></p></li>
    <li><p><strong>Low</strong></p></li>
    <li><p><strong>Informational</strong></p></li>
    <li><p><strong>Undefined</strong></p></li>
    </ul></td>
    </tr>
    </tbody>
    </table>

5.  Enable one or more of the following notification methods.

    <table>
    <colgroup>
    <col style="width: 21%" />
    <col style="width: 79%" />
    </colgroup>
    <thead>
    <tr>
    <th><p>Method</p></th>
    <th><p>Description</p></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><p>Email message</p></td>
    <td><p>To customize the email notification template, use supported token variables or modify the text in the <strong>Subject</strong> and <strong>Message</strong> fields.</p>
    <p>For more information, see <a href="trend-vision-one-standard-token-variables">Standard Token Variables</a> and <a href="trend-vision-one-dlp-token-variables">Data Loss Prevention Token Variables</a>.</p></td>
    </tr>
    </tbody>
    </table>

6.  To test if recipients can receive the event notification, click **Test**.

7.  Click **Save**.
:::
