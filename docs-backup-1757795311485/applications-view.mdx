# Applications view

Gain insights into the risk profile of your company's applications in Cyber Risk Overview.

The **Applications** view of **Cyber Risk Overview** provides an overview of available data about the security posture of your company's applications, including:

- Public cloud apps

- Local apps

- Connected SaaS apps

:::note

The **Applications** view is only available for users with the **Applications** asset visibility scope.
:::
