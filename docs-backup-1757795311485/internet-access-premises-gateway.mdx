# Enhancement to Internet Access On-Premises Gateway

If your organization uses a third-party proxy server to access the internet, you can now configure the proxy server as the upstream proxy to connect your deployed Internet Access On-Premises Gateway to the internet.

For details on configuring upstream proxy for an Internet Access On-Premises Gateway, see [Deploying an Internet Access On-Premises Gateway](deploy-ia-premises-gateway.mdx).

**Zero Trust Secure Access → Secure Access Configuration → Internet Access and AI Service Access Configuration**
