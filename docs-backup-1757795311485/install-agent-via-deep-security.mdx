# Install Trend Vision One Endpoint Security agent via Deep Security Agent

Trend Vision One Server & Workload Protection allows you to install Trend Vision One Endpoint Security agent via Deep Security Agent.

## Before you begin {#before-you-begin}

Before you begin:

### Procedure {#procedure}

1.  Verify that you are installing from a [supported platform](https://docs.trendmicro.com/en-us/documentation/article/trend-vision-one-agent-system-requirements) and Deep Security Agent version 20.0.1.25770 (20 LTS Update 2024-12-10) or later.

2.  [Update Trend Vision One to the Foundation Services release](https://docs.trendmicro.com/en-us/documentation/article/trend-vision-one-update-foundation-services-release).

3.  Configure Trend Vision One [Proxy settings](proxy-settings.mdx) to allow access to the installation package.

4.  [Configure a proxy connection and review the firewall requirements](setup-endpoint-security-for-new.mdx).

5.  Check if you need to use Trend Vision One Endpoint Sensor.

### Next steps {#next-steps}

You are now ready to install your Trend Vision One Endpoint Security agent.

## Install Trend Vision One Endpoint Security agent {#install-trend-vision-one-endpoint-security-agent}

### Procedure {#procedure-1}

1.  In the Workload Security console, go to **Computers**, and then do one of the following:

    1.  Right-click the computers on which you want to install Trend Vision One Endpoint Security agent, and then select **Actions \> Install Trend Vision One Endpoint Security agent**.

        :::tip

        If "Install Trend Vision One Endpoint Security agent" does not appear in the **Actions** menu, add the "Trend Vision One Endpoint Security Agent readiness status" column (from **Computers** \> **Columns**).
        :::

    2.  Select the computers that you want to upgrade, click **Actions** at the top, and then select **Install Trend Vision One Endpoint Security agent**.

2.  Click **Submit**.

## Schedule a task {#schedule-a-task}

To install Trend Vision One Endpoint Security agent automatically, you can [create a scheduled task](schedule-perform-tasks.mdx).

## Use Trend Vision One Endpoint Sensor {#use-trend-vision-one-endpoint-sensor}

To enhance Activity Monitoring protection using the Endpoint Sensor, consult the [Endpoint Security Policies](sensor-settings-policies.mdx)) before proceeding with the Trend Vision One Endpoint Security agent installation.

:::note

When Trend Vision One Endpoint Sensor is enabled, Activity Monitoring is automatically disabled. Activity Monitoring does not need to be disabled manually.
:::
