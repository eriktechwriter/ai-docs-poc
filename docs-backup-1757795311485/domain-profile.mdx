# Domain profile

View detailed information about a selected domain and how the domain affects the Cyber Risk Index.

The following table describes the tabs available on the domain profile screen.

:::warning[Important]

Customers with XDR sensor entitlements that have not allocated credits to Cyber Risk Exposure Management can only access the **Risk Assessment**, **Asset Graph**, **Devices**, and **Asset Profile** tabs.
:::
