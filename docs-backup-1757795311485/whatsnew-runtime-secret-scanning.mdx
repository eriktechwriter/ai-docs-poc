# Runtime Secret Scanning now available

May 31, 2025—Trend Vision One now supports Runtime Secret Scanning in Container Security. Enable Runtime Secret Scanning to detect secrets in your containers, ensuring that any exposures in production containers are detected quickly.

For more information, see [Enable Runtime Security and scanning features](runtime-security-scanning-kubernete.mdx).

**Cloud Security → Container Security → Container Inventory**
