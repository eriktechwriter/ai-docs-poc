# Invalid account error upon console logon

Problem: An error message indicating the invalid logon account appears when you log on to the Cloud Email and Collaboration Protection management console.

The possible reason is as follows:

You have created multiple CLP accounts under one CLP company ID, and you try to log on to the Cloud Email and Collaboration Protection management console not using the primary CLP account.

Perform the following:

Use the primary CLP account to log on to the management console.
:::
