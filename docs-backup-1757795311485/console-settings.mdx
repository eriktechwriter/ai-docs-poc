<td>Timeout</td>
<td><p>Allows organizations to customize inactivity timeout and max session timeout to comply with regulatory standards or internal security policy.</p>
<p><strong>Inactivity timeout</strong> is the amount of time a user can be inactive before being signed out. Its value can range from 5 minutes to 8 hours. Its default value is 15 minutes. The inactivity timeout is renewed whenever cursor or keyboard input is detected.</p>
<p><strong>Max session timeout</strong> is the amount of time before a user is signed out and re-authentication is required to continue. Its value can range from 30 minutes to 8 hours. Its default value is 8 hours. The max session timeout cannot be renewed, but the user will be notified 5 minutes and 1 minute before it expires.</p>


:::note

<p>Inactivity timeout cannot exceed max session timeout.</p>
