# Configuring Log Aggregation

Log aggregation allows you to conserve network bandwidth by sending only selected data from managed products to the Trend Micro Apex Central server.

:::danger[Warning]

Trend Micro Apex Central cannot recover data that managed products do not send to the Trend Micro Apex Central server.
:::

### Procedure {#procedure}

1.  Go to **Detections → Logs → Log Aggregation Settings**.

    The **Edit Log Aggregation Rule** screen appears.

2.  Select **Enable log aggregation**.

3.  Expand log categories.

4.  Clear check boxes to stop sending data from managed products to Trend Micro Apex Central.

5.  Click **Save**.
