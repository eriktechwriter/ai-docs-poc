# Timeline

Create investigation timelines with the collected evidence to gain insight into the context of an incident.

The following table outlines the actions available in the **Timeline** screen.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>

<th><p>Action</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Filter timeline elements</p></td>
<td><p>Specify the operating system to locate specific timeline entries.</p>
<ul>
<li><p><strong>Timestamp</strong></p></li>
<li><p><strong>Timestamp type</strong></p></li>
<li><p><strong>Evidence type</strong></p></li>
<li><p><strong>Endpoint</strong></p></li>
</ul></td>
</tr>
<tr>
<td><p>Add a custom element to the timeline</p></td>
<td><p>Click <strong>Add Element</strong> (<img src="./images/add_icon=cf892c2f-1a1f-4d22-848f-023067e4a507.webp" />) to add a custom element to the timeline.</p></td>
</tr>
<tr>
<td><p>Delete elements from the timeline</p></td>
<td><p>Select one or more elements from the timeline and then click <strong>Delete Selected</strong>.</p></td>
</tr>
<tr>
<td><p>Reset timeline</p></td>
<td><p>Click <img src="./images/options=ddb0b67f-0654-4aa5-8bc7-48ec554c5448.webp" /> and select <strong>Delete All</strong> to delete all elements from a timeline.</p></td>
</tr>
<tr>
<td><p>Add timeline to sub-case</p></td>
<td><p>Click <img src="./images/options=ddb0b67f-0654-4aa5-8bc7-48ec554c5448.webp" /> and select <strong>Add Timeline to Case</strong> to add the current timeline to the specified Forensics sub-case.</p></td>
</tr>
