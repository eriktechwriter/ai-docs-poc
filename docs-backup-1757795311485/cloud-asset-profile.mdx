# Cloud asset profile

View detailed information about a selected cloud asset.

The following table describes the tabs available on the cloud asset profile screen.

:::warning[Important]

Customers with XDR sensor entitlements that have not allocated credits to Cyber Risk Exposure Management can only see the **Risk Score** widget of the **Risk Assessment** tab and access the **Asset Profile** tab.
:::

<td><p>A cloud asset's risk assessment displays activity in several widgets.</p>

<li><p><strong>Risk Score</strong>: Assessment of the cloud asset's activity over the last 30 days</p>


:::note

<p>A cloud asset's risk score is calculated every 4 hours.</p>


:::

</li>
<li><p><strong>Risk Indicators</strong>: A list of events that might indicate the cloud asset is vulnerable to attack</p>
<ul>
<li>Expand a risk indicator to view any available <a href="trend-vision-one-attack-prevention">attack prevention/detection rules</a> that can help to mitigate the risk.</li>
</ul></li>
</ul></td>
</tr>
<tr>
<td><p>Asset Risk Graph</p></td>
<td><p>The Asset Risk Graph illustrates the asset's connection to other assets in your environment.</p>
<ul>
<li><p>Click any asset in the Asset Risk Graph to view details, including asset criticality, significant profile tags, and risk indicators.</p></li>
<li><p>Click the label in an arrow to view a description of the relationship, such as the first and last time a connection occurred.</p></li>
</ul>


:::warning[Important]

<p>The Asset Risk Graph is currently only available for some cloud asset types. For unsupported types, the <strong>Asset Risk Graph</strong> tab is inaccessible and displays "Coming Soon". Support for additional cloud asset types is under development.</p>


:::


:::tip

<p>Click a group to see a list of the cloud assets that comprise the group.</p>


:::

You may modify the values for select pre-defined tags by clicking the edit (<img src="./images/edit_icon=GUID-1F1D1164-5310-4D6D-ACD0-6049C86960AF.webp" />) icon next to the tag. To modify custom tags on the asset, click <strong>Manage Tags</strong></p>
<p>To manually modify the criticality of the asset, click <strong>Modify Criticality</strong>, select a criticality source of <strong>Custom</strong>, and select the criticality of the asset.</p></td>
