# Group Policy

View detailed identity information about the group policies in your organization.

The **Group Policy** tab allows you to view details of all group policies in your organization. Group policies are used to define configurations for groups of users and computers, such as the system behavior, application settings, and security settings.

The following table details the columns on the **Group** tab.
