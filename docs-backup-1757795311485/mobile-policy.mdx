# Mobile Policy

Manage mobile policies that apply to individual users or Active Directory groups.

A mobile policy is a collection of security settings that is deployed to the Mobile Agents. **Mobile Security** provides default policies for all users of Android and iOS/iPadOS devices in your organization. Policies for Chrome OS are currently unsupported.
