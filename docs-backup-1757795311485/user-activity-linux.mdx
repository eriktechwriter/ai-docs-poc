# User activity

View data and descriptions of evidence in the user activity category collected from Linux endpoints.

The following table contains descriptions of the evidence data in the user activity category that the [Collect Evidence task](collect-evidence-task.mdx) and [Trend Micro Incident Response Toolkit](manual-evidence-collection.mdx) may collect from Linux endpoints. These evidence types appear in columns after selecting an evidence category when examining an [Evidence Report](evidence-report.mdx).
