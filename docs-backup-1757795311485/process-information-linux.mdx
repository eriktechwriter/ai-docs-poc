# Process information

Learn about the types of evidence in the process information category that Forensics might collect from Linux endpoints.

### Primary evidence collected on running processes {#primary-evidence-collected-on-running-processes}

Forensics displays primary process information a table after selecting **Running Processes** in the [evidence report](evidence-report.mdx).
