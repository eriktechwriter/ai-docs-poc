# Scan Now

Scan Now is initiated remotely by administrators through the web console and can be targeted to one or several Trend Vision One Endpoint Security agent endpoints.

Configure and apply Scan Now settings to one or several Trend Vision One Endpoint Security agents and domains, or to all Trend Vision One Endpoint Security agents that the server manages.
