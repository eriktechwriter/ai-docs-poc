# Threat monitoring section

Trend Vision One™ – Cloud Risk Management offers the ability to monitor and view user activities and events within AWS account in real-time using [Real-time Threat Monitoring](real-time-threat-monitoring.mdx) (RTPM).

The Threat monitoring section will appear on *All account* or *Individual accounts* which have RTPM enabled. You can toggle RTPM access for your accounts via your subscriptions.

See [Real Time Posture Monitoring](real-time-threat-monitoring.mdx) for how to get the most out of this functionality.
:::
