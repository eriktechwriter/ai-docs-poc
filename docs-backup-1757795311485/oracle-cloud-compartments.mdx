# Oracle Cloud compartments

Manage connections with your Oracle Cloud compartments to allow Trend Vision One visibility into your cloud environment.

The Oracle Cloud screen allows you to view connected Oracle Cloud compartments and manage connections within the Cloud Accounts app. You can access the Oracle Cloud screen on the Trend Vision One console by navigating to **Cloud Security → Cloud Accounts → Oracle** and clicking on the **Oracle** tab.

You can take the following actions from the Cloud Accounts - Oracle screen.
