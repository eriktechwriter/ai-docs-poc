# New names for the Search app and XDR Threat Investigation

August 4, 2025—To align with the expanding scope of XDR capabilities, the Search app has been renamed to **XDR Data Explorer** and XDR Threat Investigation app group has been renamed to **Agentic SIEM & XDR**.

For more information, see [XDR Data Explorer](search-app.mdx).
