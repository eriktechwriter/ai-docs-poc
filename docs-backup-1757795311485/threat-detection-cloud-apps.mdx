# Threat detection

Learn about the widgets available on the **Threat Detection** tab.

The **Threat Detection** tab groups widgets into different sections, each providing a different perspective about threats in your organization. You can configure how this tab displays threat detection data. For details, see [Viewing threat detection data](viewing-threat-detection-data.mdx).
