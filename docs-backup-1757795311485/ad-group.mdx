# Group

View details about the groups in your organization.

The **Group** tab allows you to view details of all groups in your organization. Groups organize users or computers to simplify access management.

The following table details the columns on the **Group** tab.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>

<tr>
<th><p>Column name</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Group policy name</p></td>
<td><p>Name of a group policy in your organization.</p></td>
</tr>
<tr>
<td><p>Privileged</p></td>
<td><p>Indicates whether a group is privileged.</p>
<p>For details about default privileged security groups, see <a href="trend-vision-one-ad-privileged-security-groups">Active Directory (on-premises) default privileged security groups</a>.</p></td>
</tr>
<tr>
<td><p>Group type</p></td>
<td><p>Type of a group, which defines whether a group is security enabled.</p>
<ul>
<li><p>Security group: provides a logical grouping of objects and can be used for access control</p></li>
<li><p>Distribution group: used for grouping purposes only, such as for sending email to a collection of users</p></li>
</ul></td>
</tr>
<tr>
<td><p>Scope</p></td>

</ul>

<td><p>A fully qualified unique name that identifies a group in a directory.</p></td>
</tr>
