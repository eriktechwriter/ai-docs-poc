# Scheduled Reports

The **Scheduled Reports** screen provides a list of all reports that automatically generate on a user-defined schedule. You can use this screen to view basic information about previously configured scheduled reports, add new scheduled reports, and enable/disable scheduled reports.

The following table outlines the available tasks on the **Scheduled Reports** screen.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Task</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Add new scheduled report profiles</p></td>
<td><p>Click <strong>Add</strong> to create a new scheduled report profile.</p>
<p>For more information, see <a href="trend-vision-one-adding-scheduled-reports">Adding Scheduled Reports</a>.</p></td>
</tr>
<tr>
<td><p>Edit scheduled report profiles</p></td>
<td><p>Click the <em>Name</em> of an existing scheduled report profile to edit.</p>
<p>For more information, see <a href="trend-vision-one-editing-scheduled-reports">Editing Scheduled Reports</a>.</p></td>
</tr>
<tr>
<td><p>Copy scheduled report profiles</p></td>
<td><p>Select one or more existing scheduled report profiles and click <strong>Copy</strong> to replicate the selected profiles.</p>
<p>Click the <em>Name</em> of a copied scheduled report profile to edit.</p>
<p>For more information, see <a href="trend-vision-one-editing-scheduled-reports">Editing Scheduled Reports</a>.</p></td>
</tr>
<tr>
<td><p>Delete scheduled report profiles</p></td>
<td><p>Select existing scheduled report profiles and click <strong>Delete</strong>.</p></td>
</tr>
<tr>
<td><p>View previously generated scheduled reports</p></td>
<td><p>Click the <strong>View</strong> link in the <em>History</em> column for the report you want to view.</p>
<p>For more information, see <a href="trend-vision-one-viewing-scheduled-reports">Viewing Scheduled Reports</a>.</p></td>
</tr>
<tr>
<td><p>Enable or disable scheduled reports</p></td>
<td><ul>
<li><p>To disable a scheduled report, click the enabled (<img src="./images/enable=GUID-B00EF00D-D2A0-455F-949C-78C31DC49848=1=en-us=Low.webp" />) icon in the <em>Enable</em> column.</p></li>
<li><p>To enable a scheduled report, click the disabled (<img src="./images/disable=GUID-FB3E363E-6A03-4017-8E1E-86F3BD03D3DE=1=en-us=Low.webp" />) icon in the <em>Enable</em> column.</p></li>
</ul>


:::note

<p>Newly added scheduled report profiles are enabled by default.</p>


:::

</td>
</tr>
