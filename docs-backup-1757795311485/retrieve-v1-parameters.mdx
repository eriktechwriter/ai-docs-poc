# Retrieve the Trend Vision One parameters

### Procedure {#procedure}

1.  Go to **Dashboard → Endpoint Security → Endpoint Inventory**.

2.  Click **Agent Installer**.

3.  Click **Deployment Script**.

4.  Select **Endpoint Sensor** for **Protection type**.

5.  Select either **Linux** or **Windows** for **Operating system**.

6.  In **Preview script**, copy these values:

    - For Linux:

      `company_id`

      `scenario_ids` under `x86_64` (x86_64_xbc_agent_token)

      `scenario_ids` under `aarch64` (aarch64_xbc_agent_token)

    - For Windows:

      `AGENT_TOKEN_WIN64` (windows_token)
