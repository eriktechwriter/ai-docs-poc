# Ports and URLs used by Virtual Network Sensor

Use these settings to configure your firewall and connected products.

In order for Virtual Network Sensor to send data to **Network Security**, configure your product with the ports and URLs listed below.

To the full list of firewall and FQDN exceptions, see [Firewall exception requirements for Trend Vision One](firewall-exception-requirements-for.mdx).
