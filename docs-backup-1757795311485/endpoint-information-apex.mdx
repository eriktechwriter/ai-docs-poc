# Endpoint Information

The **Endpoint** information screen provides more detailed information about the selected endpoint. The **Endpoint** information screen title displays the endpoint icon (![](/images/endpoint_info_icon=GUID-AEA9F606-B23B-4A18-8BC5-CD02F7A1035A=1=en-us=Low.webp)) followed by the endpoint name.

Click one the following tabs to view related information.

- **Threats**: Displays all security threats detected on the selected endpoint

  For more information, see [Security Threats on Endpoints](threats-endpoints.mdx).

- **Policy Status**: Displays the list of policies associated with the selected endpoint

  For more information, see [Policy Status](policy-status-apex-tab.mdx).

- **Notes**: Displays any manually added notes about the selected endpoint

  For more information, see [Notes for Endpoints](notes-for-endpoints.mdx).

- **General Information**: Displays basic information regarding the selected endpoint

  For more information, see [General Information for Endpoints](general-information-for-endpoints.mdx).

The **Endpoint** information screen also allows you to take specific actions on the selected endpoint by using the **Task** menu.

- **Assign tags**: Associates tags with the selected endpoint for search purposes

  For more information, see [Custom Tags](custom-tags.mdx).

- **Isolate**: Limits the access the endpoint has to the network and Internet

  For more information, see [Isolating Endpoints](isolate-network-endpoints.mdx).

- **Restore**: Restores network access to the isolated endpoint

  For more information, see [Isolating Endpoints](isolate-network-endpoints.mdx).
