# Identity Summary

View details about each common identity-related asset type detected in your environment.

The **Identity Summary** screen provides you with an overview of the assets belonging to each common identity-related asset type detected in your environment.

The following table details the actions available in the **Identity Summary** screen.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<tbody>
<tr>
<td><p>Filter by asset type</p></td>
<td><p>To view the assets related to an identity-related asset type, select the asset type in the <strong>Type</strong> drop-down menu.</p></td>
</tr>
<tr>
<td><p>Create a <strong>Security Awareness</strong> training campaign</p></td>
<td><p>Click <strong>Create <strong>Security Awareness</strong> Training Campaign</strong> to create a training campaign targeting the user accounts currently listed on the screen.</p>


:::warning[Important]

<p>This is a "Pre-release" feature and is not considered an official release. Please review the <a href="trend-vision-one-pre-release-disclaimer">Pre-release disclaimer</a> before using the feature.</p>


:::

</td>
</tr>
<tr>
<td><p>Export list</p></td>
<td><p>Click <strong>Export</strong> to export a CSV file of the currently listed risk events.</p></td>
</tr>
<tr>
<td><p>View additional asset details</p></td>
<td><p>Click any asset name to view the details of the asset in the relevant <a href="trend-vision-one-asset-profile-screens">asset profile screen</a>.</p></td>
</tr>
</tbody>
</table>
