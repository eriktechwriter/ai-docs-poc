 Update to the latest version To ensure optimal performance and access to the latest features, update to the most recent version of Trend Micro Artifact Scanner (TMAS). Procedure Download the updated binary for the latest version of the TMAS CLI. Adjust your system's binary path settings by replacing the existing TMAS binary with the updated binary.
