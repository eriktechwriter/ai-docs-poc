 Configuring Scheduled Scan Settings Procedure Select the check box to enable Scheduled Scan. Click the Target tab to configure the general scan and CPU usage settings, and the scan schedule. For more information, see Scheduled Scan: Target Tab. Click the Action tab to configure the scan actions Apex One (Mac) performs on detected security threats. For more information, see Scheduled Scan: Action Tab.
