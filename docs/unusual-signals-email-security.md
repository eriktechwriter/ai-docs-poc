Signal Description Account-Takeover This sender account might be compromised. Unusual-URL The URLs in the email are similar to those found in other malicious emails. Payment-PDF-Free-Email This message originates from a free email service and discusses payment-related issues in a PDF attachment. Payment-HTML-Free-Email This message originates from a free email service and discusses payment-related issues in an HTML attachment. Payment-HTML-NB-Account This account has no prior contact history with you and discusses payment-related issues in an HTML attachment. Forged-Brand The sender claims to be a well-known brand.
