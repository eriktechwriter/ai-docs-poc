 Top analyzed advanced threats (files) chart The Top Analyzed Advanced Threats (Files) chart on the Top Statistics tab of Dashboard displays the email addresses that received the most messages containing advanced file threats based on the selected mail traffic direction. Note The data on this tab is displayed for incoming mail traffic only. Hover over a bar to see details. Select a time period by Date, Week, or Month to show data for the selected time period. The specified time period only affects the data shown on this chart and its corresponding thumbnail chart on the Threats tab. Changing these selections does not affect other charts.
