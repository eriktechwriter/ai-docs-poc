/*<![CDATA[*/ $('#title').html($('meta[name=map-description]').attr('content')); /*]]>*/ DLP Violated Policy Widget This widget displays the DLP violated policy. Use this widget to check the total number of DLP incidents. By default data is sorted by the number of incidents. To sort data by policy name, click the Policy column title. Use the Range drop-down to select the time period for the data that displays. Clicking links in the Incidents column opens a screen that displays detailed information.
