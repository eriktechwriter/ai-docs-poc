To access the CLI, connect to and open the Connector virtual appliance (either directly or through SSH). Log on to the CLI with your account credentials. To view basic information of the Private Access Connector, such as network settings and service status, run the show command. For example, use show ip route to check the network IP route. For a list of available commands, type show and then press the ? key. To view a list of tasks you can perform and enable administrative commands, run the enable command. Enabling administrative commands changes the command prompt from > to #. The following table lists out the administrative commands you can use to manage the Connector virtual appliance.
