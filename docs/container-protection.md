The following table provides an overview the main sections in Configuration. Section Description Policy Configure and manage policies that you can then assign to Amazon ECS and Kubernetes clusters. Compliance Enable compliance scanning to scan your Amazon EKS clusters for security compliance issues and generate reports with actionable insights. Important Compliance scanning is currently only supported on EKS clusters mapped within Trend Vision One to a connected AWS account. Real-Time Posture Monitoring must be enabled in that account, and the latest Container Security software must be installed. Object management Manage attestors and Configure rulesets and that you can apply to policies that are assigned to clusters with Runtime Scanning enabled.
