 Accessing the Cloud Email and Collaboration Protection management console The management console is the central point for monitoring Cloud Email and Collaboration Protection throughout the corporate network. The console comes with a set of default settings and values that you can configure based on your security requirements and specifications. The management console uses standard Internet technologies, such as JavaScript, Rest API, HTML and HTTPS.
