 Key Performance Indicators Widget Use this widget on the Trend Micro Apex Central Dashboard screen to display Apex One (Mac) key performance indicators (KPIs) based on selected criteria. For information on how to add a widget to the Dashboard screen, see the Trend Micro Apex Central or Control Manager documentation. Tip By default, the widget marks events as Important () at 15 occurrences and Critical () at 30 occurrences. Optionally, mark events as Important or Critical by customizing event thresholds.
