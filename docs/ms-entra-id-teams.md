/*<![CDATA[*/ $('#title').html($('meta[name=map-description]').attr('content')); /*]]>*/ Creating an Microsoft Entra ID app for Teams protection Procedure Log on to the Microsoft Entra ID portal as a Microsoft 365 Global Administrator. Register an app. Click Microsoft Entra ID, App registrations, and New registration. The Reigster an application page appears. Name the application. Select a supported account type. The account type determines who can use the app. If you have multiple organizations in Cloud Email and Collaboration Protection, Trend Micro recommends that you select Accounts in any organizational directory (Any Microsoft Entra ID directory - Multitenant) for all the organizations to use this app. Under Redirect URI, select Web and type <URL of your Trend Vision One console logon page>/ui/cas/provision.html. For example, if your URL is https://portal.xdr.trendmicro.com, type https://portal.xdr.trendmicro.com/ui/cas/provision.html. Click Register. The Overview screen of the registered app appears. Record the value of Application (client) ID. You need to use the app ID when granting Cloud Email and Collaboration Protection access to Teams Chat. Assign permissions to the app. Click the Manifest tab. Locate resourceAccess under requiredResourceAccess, change the parameter value to the following, and click Save.
