During a virus outbreak, the Trend Vision One Endpoint Security agent automatically enables Real-time Scan and you cannot disable scanning until the outbreak ends. Real-time Scan helps prevent the virus from modifying or deleting files and folders on endpoints. Configure the Target settings. For more information, see Real-time Scan: Target Tab. Configure the Action settings. For more information, see Real-time Scan: Action Tab. Configure the Scan Exclusion settings. For more information, see Real-time Scan: Scan Exclusion Tab.
