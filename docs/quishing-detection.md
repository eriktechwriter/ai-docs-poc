To protect your organization against quishing attacks, Cloud Email and Collaboration Protection can scan the QR codes in the email body and attachments for security threats. Supported Service Supported File Format Configuration Exchange Online JPG PNG BMP TIFF GIF WEBP To enable QR code scanning for these file formats, in your Advanced Threat Protection policy, enable Web Reputation and then Message Attachments in the rule settings. PDF To enable QR code scanning for this file type, enable Virtual Analyzer in your Advanced Threat Protection policy.
