 Synchronizing Contact Information with Active Directory Trend Micro Apex Central synchronizes data from the Active Directory Global Category (GC). Procedure Open the Microsoft Management Console (mmc). Add a snap-in (Active Directory Schema). In left panel, go to Attribute. Enable Replicate this attribute to Global Catalog for each of the following: proxyAddresses department homephone PhysicalDeliveryOfficeName telephoneNumber title Wait until Active Directory replication occurs.
