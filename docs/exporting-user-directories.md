/*<![CDATA[*/ $('#title').html($('meta[name=map-description]').attr('content')); /*]]>*/ Exporting user directories You can export valid recipients, groups and email aliases to a comma-separated values (CSV) file. Procedure Choose to export valid recipients, groups or email aliases and do the following: Select a domain from the Valid recipients drop-down list and click Export to CSV. Select a group from the Groups drop-down list and click Export to CSV. Next to Email aliases, click Export to CSV.
