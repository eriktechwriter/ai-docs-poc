/*<![CDATA[*/ $('#title').html($('meta[name=map-description]').attr('content')); /*]]>*/ Importing user-defined passwords A maximum of 100 passwords is allowed. Procedure In the User-Defined Passwords section, click Import. The Import Passwords dialog box appears. Next to File location, browse and select a TXT file to import. You can click Download sample file to view a sample of a properly formatted file. Cloud Email Gateway Protection checks all the entries in the selected file to identify any invalid, duplicate or conflicting passwords. After you confirm all the entries to be imported, click Import.
