 Virtual Analyzer file analysis details chart The Virtual Analyzer File Analysis Details chart on the Threat tab of Dashboard displays the number and level of file threats detected by Virtual Analyzer based on the selected mail traffic direction. Note The data on this tab is displayed for incoming mail traffic only. Select a time period by Date, Week, Month, or Last 12 months to show data for the selected time period. The specified time period only affects the data shown on this chart and its corresponding thumbnail chart on the Threats tab. Changing these selections does not affect other charts.
