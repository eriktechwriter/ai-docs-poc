 Virtual Analyzer quota usage details The Virtual Analyzer Quota Usage Details chart on the Threats tab of Dashboard displays the usage of the Virtual Analyzer submission quota. Note The data on this tab is displayed for incoming mail traffic only. Select a time period by Date, Week, Month, or Last 12 months to show data for the selected time period. The specified time period only affects the data shown on this chart and its corresponding thumbnail chart on the Threats tab. Changing these selections does not affect other charts.
