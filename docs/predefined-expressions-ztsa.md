Data loss prevention comes with a set of predefined expressions. These expressions cannot be modified or deleted. Data loss prevention verifies these expressions using pattern matching and mathematical equations. After data loss prevention matches potentially sensitive data with an expression, the data may also undergo additional verification checks.
