 Turn off scanning in AWS buckets Stop File Security Storage from scanning Amazon Web Services (AWS) buckets To stop File Security from scanning AWS buckets, turn off EventBridge. Procedure On the Inventory tab, select the buckets that you want to stop scanning. Click Turn Off EventBridge. Verify the selected buckets.
