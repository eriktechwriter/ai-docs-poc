/*<![CDATA[*/ $('#title').html($('meta[name=map-description]').attr('content')); /*]]>*/ Editing or deleting domains Procedure On the Domains screen, select domains by doing one of the following: To select one or more domains, select the check box to the left of each entry. To select all domains, select the check box to the left of the Domain Name column title. To edit information for a domain, do the following: Click the domain name in the Domain Name column. The Edit Domain screen appears, with fields pre-filled with the information on record for that domain. Modify the fields as needed. To delete domains, select one or multiple domain records and click Delete. After you delete the domain(s), related quarantined emails will be removed and can no longer be released.
