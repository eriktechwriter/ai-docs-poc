/*<![CDATA[*/ $('#title').html($('meta[name=map-description]').attr('content')); /*]]>*/ Exporting senders Cloud Email Gateway Protection allows you to export the existing approved and blocked senders to the local storage. Procedure Click the Approved Senders or Blocked Senders tab. Export senders by using either of the following methods: To export senders and recipients in pairs, select one or more recipient records, and click Export. To export all senders of a specific recipient, click the recipient name, and click Export All in the dialog box that appears. The selected senders are exported to the local storage.
