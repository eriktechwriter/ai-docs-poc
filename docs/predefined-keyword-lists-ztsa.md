 Predefined keyword lists Learn details about the predefined keyword lists used in data loss prevention templates. Data loss prevention provides a set of predefined keyword lists. These keyword lists cannot be modified or deleted. Each list has its own built-in conditions that determine if the template should trigger a policy violation.
