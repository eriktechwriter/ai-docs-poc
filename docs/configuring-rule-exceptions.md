 Configuring Rule Exceptions Add rule IDs to the Rule Exceptions list to exclude scan detections based on rule IDs. Procedure When configuring a parent policy, specify how other users can configure child policies. Inherit from parent: Child policies must use the settings configured in the parent policy Extend from parent: Child policies can append additional settings to the settings inherited from the parent policy Note If your child policies Extend from parent, you can configure Child Policy Restrictions to prevent child policies from adding specified rules to the Rule Exceptions list. Type a valid rule ID. You can find rule IDs on the Log Query screen. Click Add. The system displays a new entry in the table. To remove an entry from the list, click the Delete icon.
