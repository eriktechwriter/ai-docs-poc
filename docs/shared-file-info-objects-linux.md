The following table contains descriptions of the attributes in the shared file info objects category that the Collect Evidence task and Trend Micro Incident Response Toolkit may collect from Linux endpoints. These attributes are displayed associated with other evidence types when examining an Evidence Report. Note Shared file info objects may appear in multiple evidence categories or types.
