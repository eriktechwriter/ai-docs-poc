 Adding a Deployment Schedule You can configure a deployment schedule to deploy updated components to selected managed products based on the schedule you specify. Procedure Access the Scheduled Update or Manual Update screen. In the Deployment Plan section, select Define different deployment plans for managed products. Click + Add. The Add Schedule screen appears. Configure the deployment schedule. From the Managed Products/Folder tree, select a managed product or product folder. Click OK to save the settings. After creating a deployment plan, you can perform the following tasks: Click a schedule to edit the deployment schedule settings. Click Delete to remove a selected deployment schedule.
