 Alibaba Cloud Account Information Configure basic settings for your connected Alibaba Cloud accounts. Access the Account Information tab in the Account Settings screen to view and configure basic settings. After changing a field, click Save Changes. Updating information on the Account Information tab does not require performing a resource update. Field Description Account ID The ID of the connected account This information is automatically provided by Alibaba Cloud when connecting an account.
