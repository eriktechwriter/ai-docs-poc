When a particular scan type runs, Apex One (Mac) checks the scan exclusion list to determine which files on the endpoint will be excluded from scanning. Scan Exclusion List Details Files Apex One (Mac) will not scan a file if: The file is located under the directory path specified in the scan exclusion list The file matches the full file path (directory path and file name) specified in the scan exclusion list File extensions Apex One (Mac) will not scan a file if its file extension matches any of the extensions included in this exclusion list.
