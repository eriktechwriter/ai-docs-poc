New keyword expressions can be defined and saved, either from scratch or by copying and editing an existing expression. Procedure Create or edit a content filtering policy. Click the Scanning Criteria tab. Select Advanced and click keyword expressions for each condition. Select an existing keyword expression from the Available field. Click the move button (Add>) to move the selected keyword expression to the Selected field. Note You can also add, edit, copy, or delete keyword expressions.
