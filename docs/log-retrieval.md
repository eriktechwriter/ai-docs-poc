 Log retrieval The Log Retrieval tab allows you to decide whether to retrieve policy event logs and mail tracking logs via REST APIs for third-party SIEM application integration. To retrieve the logs, you must enable the log retrieval function first. For more information about log retrieval via REST APIs, refer to the Cloud Email Gateway Protection REST API Online Help.
