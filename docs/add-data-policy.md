/*<![CDATA[*/ $('#title').html($('meta[name=map-description]').attr('content')); /*]]>*/ Add a data policy Create a data policy, which specifies a network location containing local devices and files that you want monitored for threats and data leakage. Procedure In the Trend Vision One console, go to Data Security > Data Policy. Click Create Policy. Provide a name for the policy. In the Network location field, enter the Uniform Resource Identifier (URI) of the network location. Optionally, add a description of the data policy. To apply the policy only to files in the network location with specific data attributes: Enable Advanced Settings. Clear the checkbox beside each file type you want to exclude from the policy.
