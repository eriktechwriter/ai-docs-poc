# Cloud Risk Management Scan frequency

By default, the Cloud Risk Management Scan runs every hour, however, there is the option to modify the intervals between each bot run.

### Steps to modify Cloud Risk Management Scan frequency {#steps-to-modify-cloud-risk-management-scan-frequency}

1.  Input a value for *Delay between runs*.
2.  Click *Update Scan settings.*
3.  Once modified, the interval will be displayed on the [Settings page](cloud-acct-cloud-posture.mdx). {.zoom}

:::note

Cloud Risk Management Scan does not allow you to set up a frequency of less than an hour.
:::
::::
