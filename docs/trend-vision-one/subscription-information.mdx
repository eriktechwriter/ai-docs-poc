# Subscription Information

Configure basic settings for your connected Azure subscriptions.

The Subscription Information tab in the Subscription Settings screen allows you to view and configure basic settings. After changing a field, click **Save Changes**. Updating information on the Subscription Information tab does not require performing a resource update.
