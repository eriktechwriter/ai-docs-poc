# Using body is blank criteria

Cloud Email Gateway Protection can scan messages for blank bodies.

:::note

Cloud Email Gateway Protection detects any of the following cases as blank bodies:

- Bodies with no text nor HTML tags

- Bodies with only white space characters

- No body entity
:::

### Procedure {#procedure}

1.  On the **Scanning Criteria** tab, click **Advanced**.

2.  Select **Body is** "**blank**".
