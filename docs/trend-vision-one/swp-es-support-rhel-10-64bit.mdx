# Server & Workload Protection and Endpoint Sensor support Red Hat Enterprise Linux 10 64-bit (x86-64)

July 25, 2025—Server & Workload Protection and Endpoint Sensor now support deployment to Red Hat Enterprise Linux 10 64-bit (x86-64) endpoints.

You can select Red Hat Enterprise Linux 10 64-bit (x86-64) when deploying a new agent in the Endpoint Inventory app. For more information, see [Trend Vision One Endpoint Security agent system requirements](endpoint-agent-system-requirements.mdx).

**Endpoint Security → Endpoint & Workload Security**
