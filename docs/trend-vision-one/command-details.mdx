# Command Details

The **Command Details** screen displays the following information about an issued command.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Column Name</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Last Reported</p></td>
<td><p>The date and time when the managed product last sent a response to the Trend Micro Apex Central server</p></td>
</tr>
<tr>
<td><p>Server/Entity</p></td>
<td><p>The host name of the managed product server</p></td>
</tr>
<tr>
<td><p>Status</p></td>
<td><p>The status of the issued command</p></td>
</tr>
<tr>
<td><p>Description</p></td>
<td><p>Additional details about the command status</p></td>
</tr>
</tbody>
</table>

:::note

The **Command Details** screen refreshes every 30 seconds.
:::
