# Server & Workload Protection and Endpoint Sensor now support Debian 12 Linux OS

August 15, 2024—Server & Workload Protection and Endpoint Sensor now support deploying to Debian 12 endpoints. You can select the Debian 12 distribution when deploying a new agent in the Endpoint Inventory app. For details on supported Linux platforms, see [Trend Vision One Endpoint Security agent system requirements](endpoint-agent-system-requirements.mdx).

**Endpoint Security → Endpoint Inventory**
