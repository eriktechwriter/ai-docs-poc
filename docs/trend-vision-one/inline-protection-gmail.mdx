# Configuring inline protection settings for Gmail

### Procedure {#procedure}

1.  In Cloud Email and Collaboration Protection, go to **Policies → Global Settings → Other Settings → Inline Protection Settings for Exchange Online**.

2.  To use the message header address for matching the approved or blocked sender lists in the ATP policy, select **Message header address** under **Sender Address Match**.

    The envelope address is always used for matching the lists.
:::
