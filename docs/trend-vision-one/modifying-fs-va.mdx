# Modify a mount point

When you add the File Security Virtual Appliance to your Service Gateway, the Service Gateway appears on the File Security Inventory page. Virtual appliances can have multiple servers, and servers can have multiple mount points. From this page you can add, modify, remove, or turn off servers and mount points for a specific Virtual Appliance.

### Procedure {#procedure}

1.  In Trend Vision One go to **Cloud Security â†’ File Security**.

2.  Click **Inventory**.

3.  Click the **Virtual Appliance** node.

4.  Click the **Service Gateway** name.

    The Service Gateway Details page opens.

5.  Click **Manage Services**.

    The **Manage Services** drawer opens.

6.  Click the Edit (pencil) icon for the Virtual Appliance.

7.  Remove the specific mount point:

    1.  Click the Delete (trash can) icon for the mount point.

    2.  Click **Remove**.

8.  Click **Add**.

9.  Select the **Protocol**.

    :::note

    Depending on your protocol, you may need to enter the **Username** and **Password** to access the folder.
    :::

10. Enter the **Server IP**.

11. Enter the **Server folder path**.

12. Enable **Scanning**.

    You can enable scanning later if you want.

13. If you want to enable **Predictive Machine Learning**.

14. If you want to quarantine suspicious files:

    1.  Enable **Quarantine**.

    2.  Enter the name of the folder in which the files are to be quarantined in the **Quarantine destination** field.

        :::warning[Important]

        The **Quarantine destination** folder must be a subfolder of the Server folder path that you entered in step 9 and have write access enabled.
        :::

15. Click **Add**.

16. Click **Save**.
:::
