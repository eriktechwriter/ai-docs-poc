# CIS Benchmark scanning for Amazon EKS version 1.5.0 now available

January 29, 2025—Trend Vision One now supports compliance scanning for both [CIS benchmarks](eks-1-5-0-recommendations.mdx) for Amazon EKS versions 1.4.0 and 1.5.0 for your EKS clusters. You can now select which supported benchmark version will be used for each cluster type.

For more information, see [Compliance](compliance-scanning.mdx).

**Cloud Security → Container Security → Container Protection**
