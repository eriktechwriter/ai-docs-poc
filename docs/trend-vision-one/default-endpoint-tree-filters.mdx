# Default Endpoint Filters

By default, the **Endpoints** tree provides default filters based on the typical grouping of endpoints in an organization.

<figure>
<img src="./images/default_filters=GUID-C2DA47C9-0A69-4263-8AAF-EC3CA87D806D=1=en-us=Low.webp" />
</figure>

Expand one of the default filters and select the type of endpoint to display.

For details about the table columns and the data they contain, see [User Details](user-details-for-apex.mdx).

The following are the default filters:

- **Endpoint Type**: Servers, desktops, laptops, mobile devices, and other types

- **Operating System**: Common operating systems installed on endpoints, including Windows, Mac OS, Linux, iOS, Android, and other operating systems

- **Network Connection**: Manually isolated endpoints
