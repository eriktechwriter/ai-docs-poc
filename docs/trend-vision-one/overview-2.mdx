# Overview

Quickly review your organization's identity-related assets that might be exposed to attack and identify risk events that weaken your security posture.

Each widget of the **Overview** tab in the **Identity Security Posture** app provides insights into the various aspects of your organization's identity posture.

The following table describes the widgets in the **Overview** tab of **Identity Security Posture**.
