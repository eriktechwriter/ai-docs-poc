# Detection Rules

Manage detection rules for all connected Deep Discovery Inspector appliances.

The **Detection Rules** tab in **Monitoring/Scanning** displays a list of imported detection rules.

The following table outlines actions available in the **Detection Rules** tab.

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr>
<th><p>Action</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Enable central management of detection rules for connected Deep Discovery Inspector appliances</p></td>
<td><p>Turn on the toggle to enable central management of detection rules for all connected Deep Discovery Inspector appliances version 6.8 Service Pack 1 or later. Detection rule settings are applicable to Deep Discovery Inspector appliances only.</p></td>
</tr>
<tr>
<td><p>Filter detection rules by status</p></td>
<td><p>Use the drop-down menu to view detection rules by the selected status.</p></td>
</tr>
<tr>
<td><p>Manually enable or disable detection rules</p></td>
<td><p>Select detection rules and select the status from the drop-down menu.</p>


:::note

<p>Manually enabling detection rules might cause performance issues.</p>


:::

</td>
</tr>
</tbody>
</table>
:::
