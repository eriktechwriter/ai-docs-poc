# Protect Files in the Security Agent Installation Folder

To prevent other programs and even the user from modifying or deleting Trend Vision One Endpoint Security agent files, Apex One locks the following files in the root `<Agent installation folder>`:

- All digitally-signed files with `.exe`, `.dll`, and `.sys` extensions
