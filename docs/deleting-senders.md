 Deleting senders Cloud Email Gateway Protection allows you to delete the existing approved and blocked senders. Procedure Click the Approved Senders or Blocked Senders tab. Delete senders by using either of the following methods: To delete all senders of one or more recipients, select one or more recipient records, and Delete.
