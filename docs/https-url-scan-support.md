 HTTPS URL Scan Support HTTPS communication uses certificates to identify web servers. It encrypts data to prevent theft and eavesdropping. Although more secure, accessing websites using HTTPS still has risks. Compromised sites, even those with valid certificates, can host malware and steal personal information. In addition, certificates are relatively easy to obtain, making it easy to set up malicious web servers that use HTTPS. Enable checking of HTTPS URLs to reduce exposure to compromised and malicious sites that use HTTPS.
