To select multiple names, hold the CTRL key while selecting. You can also type a keyword in the Search field and click Search. The table refreshes with names that match the keyword. Click Add. The names move to the Approved List table. To remove names from the approved list, select the names and click Remove. To select multiple names, hold the CTRL key while selecting.
