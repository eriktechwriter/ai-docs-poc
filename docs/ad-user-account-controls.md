User account controls in Identity Inventory describe the flags added to the UserAccountControl attribute for Active Directory accounts. User account controls can expand or restrict the abilities of an account. Examine user account controls for an identity for more insight into an identity's capabilities and permissions. The following table lists user account controls that may be shown for an identity in Identity Inventory.
